81.〜てならない

接続 動て形＋ならない

　　　イ形て形＋ならない

　　　ナ形語幹＋でならない

説明 “…得不得了””非常…”

例文

急な仕事で、10年ぶりの同窓会に行けなくなり、残念でならない。
因为有紧急的工作,去不了十年的同学会,真的是很遗憾.

気のせいかもしれないが、最近上司に無視されているような気がしてならない。
也许是心理作用吧,最近总觉得好像是被上司忽视了。

大学の時もっと勉強しておけばよかったと悔やんでならない。
要是大学时多学点东西就好了,现在后悔得不得了.

注意

强调程度很高.表示情不自禁地去做某事,自己无法控制.

82.〜てはいられない

接続 動て形＋はいられない

説明 “不能…”

例文

仕事がたまっていて、三連休といっても休んではいられない。
积压了很多工作,虽然说是连续放假三天,也不能休息了.

事の真相を知った以上、もう黙ってはいられない。
既然知道了真相,就不能再沉默下去了.

来週は大事な期末試験なので、のんびりしてはいられない。
下周就是重要的期末考试了,不能再想平时那样悠闲了.

注意

表示迫于某种情况,不能再继续以前的状态了,或急于想做某件事,包含危机感或紧迫感等.

83.〜てばかりはいられない/てばかりもいられない

接続 動て形＋ばかりはいられない/ばかりもいられない

説明 表示不能总是以某种状态继续下去,”(也)不能总是”

例文

どんなに辛くても、泣いてばかりもいられない。もっと強く生きて行かなければならない。
无论多痛苦,也不能总是哭.必须更加坚强地活下去.

体の調子が悪いからといって、このまま会社を休んでばかりはいられない。
虽然说最近身体状况不太好,但不能总是这样请假不去上班。

今日の試験に勝ってからといって喜んでばかりはいられない。来週は決勝戦だ。
虽说今天赢了,但不能总是沉浸在喜悦当中.下周就是决赛了.

注意

常与表示感情的词搭配使用,包含了说话人对现状的不安,”不能总是这么下去,不能沉浸在…当中”

84.〜てはじめて

接続 動て形＋はじめて

説明 “在…之后才…”

例文

外国に住んでみてはじめて日本の治安のよさを実感した。
在国外居住之后才深切地体会到了日本的治安之好.

ビジネスの世界では、お互いの信頼関係があってはじめて商談や取引が成り立つのだ。
在商界,彼此之间建立相互的关系之后,谈判和交易才能达成.

人は病気になってはじめて健康の大切さに気づく。
人生病了之后才会意识到健康的重要性.

別れてはじめて相手のよさに気づく場合もある。
有时在分手之后才发现对方的优点.

注意

经过前项的事情之后,对过去没有在意的事情有了新的认识或感悟;或有了前项作为基础,后项才能得以实现.前项是后项发生的理由或者契机.

79.〜てまたらない

接続 動て形＋たまらない

　　　イ形て形＋たまらない

　　　ナ形語幹＋でたまらない

説明 “…得受不了””非常…”

例文

海外出張中は、幼い子どもたちにあいたくてたまらなかった。
在国外出差的时候,非常想念年幼的孩子们.

みんなの前で、英語の発音の間違いを指摘されて、恥ずかしくてたまらなかった。
在众人面前被指出了英语发音的错误,真是羞愧难当。

蚊に刺された脚が痒くてたまらない。
脚被蚊子叮了,奇痒难忍.

注意

①与「〜てしょうがない」意思大致相同,强调忍受不了某种感觉.不能接在表示自发的动词或词组之后,如「思える/気になる」.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等.

80.〜てでも

接続 動て形＋でも

説明 表示采取强硬的手段,”就算…也要…”

例文

今日は、徹夜してでもこの仕事を終らせなければならない。
今天就算通宵也要把这个工作做完.

姉は昔から「欲しいものは借金をしてでも買う」タイプの人だ。
我姐姐以前开始就是”想要的东西,就算是借钱也要买的”那种人.

家を売ってでも子どもの手術の費用を用意したい。
就算是卖房子,也要凑齐孩子的手术费用.

注意

表示为了达到某一目的而采取强硬的手段,后项表达了说话人强烈的愿望和意志.

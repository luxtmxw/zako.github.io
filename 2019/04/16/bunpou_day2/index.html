1.〜以上(は)

接続　動/ナ形/イ形の名詞修飾形＋以上(は)

ナ形語幹/名＋である＋以上(は)

説明　前项表示确定的事实,”既然…就…”

例文

京都に来た以上、絶対金閣寺を見に行ったほうがいいよ。
既然来个京都,就最好去金阁寺看看.

みんなの前で一人でやると宣言した以上は、誰かに手伝ってもらうことはできない。
既然在大家面前信誓旦旦地说要一个人做,就不能请别人帮忙.

学生である以上、学業を第一に考えるべきだ。
既然是学生,就应该把学业放在第一位.

注意

后项常接表示决心,义务,建议,推量等表达说话人意志的内容.

2.〜一方/一方で(は)

接続　動/ナ形/イ形の名詞修飾形＋一方/一方で(は)

ナ形語幹/名＋である＋一方/一方で(は)

説明　“一方面…另一方面…”,”一边…一边…”

例文

労働力が不足している現象について、女性をフルに活用すべきだと主張する人がいる。一方で、女性の社会進出は少子化の原因たと考えている人もいる。
关于劳动力不足的问题,有人主张应该充分利用女性劳动力.而另一方面也有人认为女性进入社会工作是导致少子化的原因.

科学の発展は便利な生活をもたらす一方で深刻な環境汚染を引き起こした。
科学的发展一方面给生活带来了便利,另一方面又引起了环境污染.

彼女は歌手として活躍する一方で、女優としても。頭角を現した。
她一方面活跃与歌坛,另一方面作为演员也崭露了头角.

注意

①前后项可以是相反的,对比的关系(如第一和第二个例句),也可以是并列的关系(如第三个例句).
②有接续词的功能,可用于连接两个句子(如第一个例句)

3.〜一方だ

接続　動辞書形＋一方だ

説明　“不断…”,”越来越…”

例文

去年の日本のGDP成長率は予想を上回った。それに今年度の第一四半期も連続で改善したので、景気回復への期待は高まる一方だ。
去年的日本GDP增长超出了预期.而且本年度第一季度也是持续增长的,所以人们对经济复苏的预期越来越高.

最近の調査によれば、未成年の犯罪率は高くなる一方だ。
根据最近的调查显示,未成年人的犯罪率越来越高.

ここ数年、経済回復のきざしが見えず、就職状況は厳しくなる一方だ。
这几年,经济没有复苏的迹象,就业形势也越来越严峻.

注意

表示某种倾向,状态等朝一个方向不断发展.

111.〜に限る

接続 動辞書形・ない形ーない/名＋に限る

説明 “最好是…”

例文

仕事で疲れた時は、お風呂上がりの冷えたビールに限ると父がよく言っていた。
爸爸经常说,工作累了的时候,泡完澡之后喝上一杯冰镇啤酒是最好的。

風邪を引いたら、薬を飲んでゆっくり休むに限ります。
感冒的话,最好吃完药后好好地休息。

寒い日は、熱々の鍋に限る。
在寒冷的日子里,最好吃热腾腾的火锅是最好的。

注意

相当于「〜のが一番いい」,一般是被普遍认可的事情

112.〜にかけては/にかけても

接続 名＋にかけては/にかけても

説明　“在…方面”

例文

性能はともかくとして、デザインにかけては、このカメラが一番だと思う。
性能暂且不说,在设计方面,我觉得这台相机是最好的.

彼女は職場ではあまり目立たないが、仕事の速さにかけては定評がある。
她在单位里不太起眼,但是工作速度快是得到了公认。

彼は数学家だが、音楽にかけても優れた才能に恵まれている。
他是一个数学家,但在音乐方面也拥有卓越的才能.

注意

相当于「〜について言えば、一番いい」。后项一般是说话人对某人或某物的正面评价,表示”某人或某物在…方面是最优秀的,最好的”

113.〜に決まっている

接続　動/イ形の普通形＋に決まっている

ナ形語幹/名＋に決まっている

説明 表示比较确切的推测,”一定…””肯定…”

例文

アマチュアがプロの選手と対決したって、負けるに決まっている。
业余选手去和专业选手较量,那肯定会输.

必修科目の単位が足りないから、卒業できないに決まっている。
必修课的学分不够,所以肯定不能毕业。

昨夜、遅くまでゲームをしていたのだから、昼間は眠たいに決まっている。
昨晚玩游戏玩到很晚,所以白天肯定很困.

注意

类似于「〜に違いない」。常用于口语.

114.〜に加えて/に加え

接続 名＋に加えて/に加え

説明 表示添加, “加上…”,”而且…”

例文

夏の記録的な高温に加えて、雨が少なかったため、今年の米の収穫量は大幅に減った。
夏季创纪录的高温,加上降雨稀少,今年大米的收成大幅度减少了。

のどの痛みに加えて、咳や熱も出てきたので会社を休むしかない。
喉咙痛,还咳嗽,发烧,所以只好向公司请假了。

原油の高騰に加えて、農産物も値上がりした。
原油价格暴涨,农作物的价格也上涨了。

注意

「〜に加え」比「〜に加えて」更书面化,两个都用在句中中顿.常常和「〜も」呼应使用,构成「〜に加えて、〜も」。

115.〜に越したことはない

接続 動辞書形/ナ形語幹(である)/イ形＋に越したことはない

名＋である＋に越したことはない

説明　“最好是…””莫过于…””…(的话)就再好不过了”

例文

一泊二日の出張だが、工事現場の視察もあるから、着替えの服は2セット用意するに越したことはない。
虽然是两天一夜的出差,但是要去施工现场视察,所以最好是准备两套换洗衣服.

申請書の提出締め切りは明日の午後四時だが、早めに出せばそれに越したことはない。
提交申请的截止日期是明天下午四点,如果可以提早交的话那就再好不过了。

子どもが無事に生まれてくれればそれに越したことはない。
孩子可以平安出生就再好不过啦.

注意

「それに越したことはない」表示”那样就再好不过了”

116.〜に応えて/に応え

接続　名＋に応えて/に応え

説明 表示响应,报答,”响应…””应…”

例文

お客様のご要望に応えて、来月から営業時間を１時間を延長させていただきます。
应客户的要求,从下个开始将延长一个小时的营业时间.

顧客の声に応えて、入会手続きはウェブでもできるようになりました。
应客户的要求,加入会员的手续费也可以在网上办理了。

当院は、患者さんのご要望に応えて、病状をわかりやすく説明することを心がけています。
本院应患者的要求,平时注意将病情解释的通俗易懂.

両親の期待に応えようとして、必死でピアノの練習をしている。
为了不辜负父母的期待,努力练习钢琴。

注意

常接在「要望/要求/期待」等词的后面,表示响应这些要求或实现这些期待.「〜に応えて/に応え」用于句中中顿.

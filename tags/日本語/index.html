第４８課

文法

1.お＋一类动/二类动します　ご＋三类动的汉子部分します 自谦语

用在自己身上

ます形 お〜します
一类动词 話します 　お話しします
二类动词 見せます 　お見せします
ます形 ご〜します
三类动词 連絡します 　ご連絡します

お荷物は私がお持ちします。

2.自谦语的特殊形式

明日、私がそちらへ伺います。(明天我去您那儿拜访.)
中国の北京から参りました。(我来自中国的北京.)
コピーは私がいたします。(复印的是我来做.)
黄教授に論文を見ていただきました。(我请黄教授给我看了论文.)
案内状、拝見いたしました。(开业通知,我们已经拜读了.)

3.动ていただけますか 请求/动ていただけませんか　＝　动てもらえませんか

用于请求对方为字做某事

この服、ちょっと小さいので、取り替えていただけませんか。(这件衣服小了点儿,能给我换一下吗)

4.使役+せていただきます　＝　します

请允许我做某事 使用频率较高,自谦程度高于”お/ご〜します”.其构成方式在动词使役形式的”て形”后面加”いただきます”

早速、資料を届け出せていただきます。(我这就把资料给您送过去)
来週の水曜日、休ませていただきませんか。(下星期三,我想休息下,行吗?)

5.ございます＝あります/ 名でございます＝です

社長、一つお伺いしたいことがございます。(经理我有个问题像您请教下)
あのう、お客様、おつりでございます。(对不起,这是找您的钱)

表达及词语讲解

1.名称的敬语形式

①名词尊他语

家を建てられたそうですね、お若いのに、立派ですね。(听说您盖了自己的房子了,您还这么年轻,真能干啊.)

②名词自谦语

帰ったら、すぐ社の者に資料を送らせます。(回去以后,马上让公司的人给您寄资料.)

2.おります/〜ております

“います”的更谦和的说法是”おります”. 相应的”ています”的更谦和的说法是”ております”

いつもお世話になっております。(承蒙多方关照)

第４７課

文法

1.尊他语

加在尊敬对象的动作行为上来
用于长辈,上级,关系疏远的人上

2.动(ら)れます　被动(自带尊他)

明日社長が来られるそうだよ。(对社长的尊敬)
部長は毎朝公園を散歩されるそうです。(据说部长每天早上都在公园散步)

3.お＋(一类动/二类动)になります　

“お+动词ます形去掉ます+になります”但是单音节不能用于在此形式,比如:見ます、寝ます、います。三类动词也不能用

社長はもう帰りましたか。
尊他用法①帰られましたか。
尊他用法②お帰りになりましたか。

旅行の日程はもうお決めになりましたか。(旅行的日程您已经定下来了吗)

4.お＋(一类动/二类动)ください。ご＋(三类动的汉子部分)ください

“お+动词ます形去掉ます+ください”.”ご+三类动词汉字部分+ください”.去掉后只有一个音的动词不能使用
エスカレーターをご利用ください。(请利用电梯.)
こちらのドアからお入りください。(请从这个门进去.)

5.尊他语的特殊形式

“知っています”的尊他语是”こ存じです”

先生、何を召し上がりますか。(老师,您吃点什么?)
まず、ホテルにチェックインなさいますか。(先到宾馆办理入住手续吗?)

第４６課

文法

1.表示比喻 名+の/动(简体形)ようです　名/动(简体形)みたいです

表示比喻时多与副词”まるで”呼应使用

よう＋な+名(人)になりたい
よう＋に＋动 いい成績を取りたい。

これは柔らかくて、まるで本物の毛皮のようです。(这个软软乎乎的,好像真皮一样)
まるでヨーロッパの町並みを見ているようですね。(真像看到了欧洲的街道)

まるでテレビドラマのような出来事ですね。(真好像是电视剧里发生的事情一样)

2.名1+らしい+名词2　典型性 具有某个事物的典型特征

男らしい(男子汉)　女らしい(有女人味)
男っぽい(女汉子)　女っぽい(娘炮)
お前らしいなぁ。(真像你做事风格)

多与副词”いかにも”(典型的,确实)

この着物はいかにも日本らしい柄ですね。(这件和服的花纹是典型的日本式)

如果”名词1”与”名词2”为同一个名词时,意思是”真正意义上的~,像样的~”

このごろ忙しくて、食事らしい食事をしていません。(最近很忙,没正经吃过一顿饭)

3.名(时间)までに

まで和までに的区别
まで: 持续一段时间用
までに: 时间点
５時から１０時までずっとゲームをしていました。
１０時までに宿題を提出してください。(请10点前提交作业)

4.名+の/动(简体形)+間(に)

“~ 間”表示某种行为或事态的持续状态,多与副词”ずっと”一起使用
学校が休みの間、アルバイトをするつまりなんだ。(我打算在学校放假的时候打工)

“~ 間に”表示某种行为或事态发生或完成的时间范围.
私が留学している間に、家の周りもずいぶん変わりました。(我留学期间,我家那一带发生了很大的变化)

5.名のような味/匂いがします　

感觉到某种味道或气味时用”味/匂いがします(有味道/气味)”
この牛乳、変な味がするけど、いつ買ったの？(这牛奶有股怪味儿,你什么时候买的?)

“〜のような味/匂いがします”的形式
この野菜、レモンのような味がしますね。
このスープは、日本の味噌汁のような匂いがします。

除了”味/匂い”之外表示听觉”声、おと”,表示感觉”感じ等,也用”~がします”形式(有声音/有感觉)
ここには昔来たことがある感じがします。(我觉得以前来过这里)
注意 这里的”~します”绝不能用成”~あります”

参考: 表示身体状况的词语也可以加在”がします”的前面使用.
さっきから、ちょっと吐き気がするんです。(从刚刚开始,觉得有点儿恶心)

第４５課

文法

1.动 + ていきます/てきました　(持续/变化)

时间和空间的变化

少子化が進んで、日本の人口はだんだん減っていくでしょう。(少子化的现象日益加剧,日本的人口将会越来越少吧)
二十年間ずっと日本語を勉強してきました。(这20年来我一直在学习日语)

2. 动 + てきました　(出现)

表示某种状态的开始和出现 只限前接非意志动词

ずっと本を読んでいたので、目が疲れてきました。(之前一直在看书,眼睛开始累了.)
お腹が空いてきたんですが、そろそろ食事に行きませんか。　(肚子有点饿了,咱们去吃饭好吗?)

3.小句1 + し, 小句2 + し, 小句3.

表示有多个理由 名词或二类形 + だ ＋ し
只有一个し　暗示还有其他的理由

美味しいし、手軽だし、私は冷凍食品をよく食べます。(好吃又简单,我经常吃冷冻食品)
電気も消えているし、かぎもかかっているし、きっと留守ですよ。(黑着灯,又锁着门,肯定不在家)

注意: “て”形和”~し、~し、~”的区别有两点:
①”て”一般只用于连接形容词和名词,而”~し、~し、~”还可以接动词
②”~し、~し、~”一般用于陈述原因,理由,而”て”形则可用于多个场合

4.动/一类形+ば　动/一类形+ほど+小句

动词ば形+动词基本形+ほど+小句　一类形容词ば形+一类形容词+ほど+小句
越来越怎么怎么样

この本は読めば読むほど、面白いです。(这本书越读越有趣.)
結婚式のスピーチは、短けば短いほどいいと言われていますね。(大家都说结婚典礼上的致辞越短越好.)

二类形容词有时也可以用于这一句型.其接续方式为:”二类形容词+なら＋二类形容词+な＋ほど”或”二类形容词+であれば＋二类形容词+な＋ほど”。前者多用于口语,后者多用于书面语.

子供は元気なら元気なほどいいです。(小孩越健康越好)
子供は元気であれば元気なほどいいです。(小孩越健康越好)

5.”で/へ/から/まで/と” + の

に后面不能加の

空港から市内までのアクセスがよくなりました。(从机场到市区的交通方便了.)
北京から東京への便は1日に１０便以上あります。(一天有10个以上的航班从北京开往东京)
これは母へのプレゼントです。
これは母にのプレゼントです。(X)

第４４課

文法

1. 表示推测 (小句简体形/名+の)ようです(みたいです口语形式) 、 (小句简体形)らしいです 和 そうです的区别

①ようです 根据自己感受到的推测(第六感也算) 主观 直接被观察,被推测

玄関のところに誰かいるようです。

風邪を引いたようです。

遠くから何か飛んできます。飛行機のようですね。
ーーそのようですね。
什么东西从远处飞过来了,好像是飞机.—-好像是.

②らしいです 当别人的面不能使用 偏客观事实

電気が消えています。李さんは出かけたらしいです。
电灯关着,小李大概是出去了.

③有时前两者可以共用 如果站在第三者的角度客观地叙述推测的结论时可共用

李さんは出かけたらしいです。
李さんは出かけたようです。

風邪を引いたようです。
風邪をひいたらしいです。(X) 错误

④(动词ます形去掉ます 或 形容词)そうです 除了推测还有 表示样态(33课)

一类形容词去掉い 加 そうです。　二类和名词直接加

このケーキはとても美味しそうです。(样态)
这个蛋糕看上去很好吃.

雨が降りそうです。
好像要下雨

注意点:
前接形容词否定时用”~そうではありません”, 而前接动词否定时则用”~そうにありません”

あまり美味しそうではありません。
雨やめそうにありません。
雨看上去不像要停的样子.

ない　なさそうです
いい　よさそうです

“~そうです”可以不用再句尾,还可以用于名词的动词前面, 用于动词前用”~そうに”,用于名词前用”~そうな”

美味しそうなケーキですね。
子供たちは楽しそうに遊んでいます。

2.小句(简体形)+らしいです 和 小句(简体形)+そうです(32课) 传闻(听说)

基本一样

天気予報によると、明日は雨だそうです。
据天气预报说明天有雨

林さんはお酒が好きらしいですよ。
据说小林喜欢喝酒.

3.动词/一类形/二类型+ 過ぎます　表示 过了(过多)

表示某种动作或事物的性质等超过了正常的量和程度. 其接续:
动词: ます形去掉ます + 過ぎます
一类形: 去掉い + 過ぎます
二类形: 直接 + 過ぎます
　
昼ご飯を食べ過ぎました。
このスープは熱すぎませんか。(你不觉得这个汤太烫了吗?)
この説明は複雑すぎます。 (这个说明太复杂了)

4.一类形 + さ

“一类形 + さ”　构成一种表示某种状态的名词. 方式吧词尾的い改成さ

この暑さはいつまで続くんでしょう。(这种炎热的天气要持续到什么时候呢)
富士山の美しさが心に残っています。(富士山的美留在了我的心底)

部分一类形后续接さ,成为表示大小,长短,轻重的名词

一类形 ~さ
大きい 大きさ　大小
高い 高さ　高度
深い 深さ　深度
長い 長さ　长度
広い 広さ　面积
太い 太さ　粗度
重い 重さ　重量
厚い 厚さ　厚度
速い 速さ　速度

某些二类形也可以后续接さ。但不如一类形用的普遍.例如:
便利さ(方便程度)　複雑さ(复杂程度)
但不能说 きれいさ(X)　ひまさ(X)

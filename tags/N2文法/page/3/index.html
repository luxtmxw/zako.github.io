91.〜どころか

接続 動/ナ形/イ形の名詞修飾形＋どころか

　　　名＋どころか

説明 表示从根本上进行否定,”哪里…””别说…”

例文

風邪だと思って市販の風邪薬を飲み続けたら、症状はよくなるどころか、悪化してしまった。
以为是感冒,所以一直在吃市面上卖的感冒药,结果病情别说好转了,反而更加恶化了.

足が痛くて、歩くどころか立つのも無理だ。
脚疼,别说走路了,连站都站不了。

この料理は美味しいどころか、辛すぎて全然食べられない。
这个料理哪里好吃啊,太辣了,根本没法吃。

注意

后项内容与前项叙述的事实相反,从根本上否定了前项,并且提出程度更深的后项,”别说…就连…也…”

92.〜ところだった

接続 動辞書形＋ところだった

説明 “差一点就…””险些…”

例文

危ない！今、電柱にぶつかるところだったでしょう。スマホなんか見ないで、ちゃんと歩いて。
危险!你刚刚差点就撞上电线杆了.别看智能手机了,好好走路.

床が濡れていることに気づかず、もう少しで倒れるところだった。
没注意到地板湿了,差点就摔倒了。

気がつくのが遅かったら、危うく火事になるところだったよ。
如果发现晚了,险些就酿成火灾了。

注意

可以用「もう少しで/危うく」等来加强语气,表示”差一点就…”.

93.〜どころではない/どころではなく

接続 動辞書形・ている形/名＋どころではない

説明 表示不是做某事的场合,”不是…的时候””哪能…”

例文

子どもが生まれる前は料理教室に通ったり、映画を見に行ったりしていた。今はそれどころではなく、育児と家事に追われる日々だ。
生孩子之前还上上烹饪班,看看电影.现在哪里有那个闲工夫,每天都忙于带小孩和做家务.

せっかく家族で海に行ったのに、強風で泳ぐどころではなかった。
难得全家人一起去海边玩,却因为风太大根本没办法游泳。

車を買ったばかりでお金がないので、海外旅行どころではない。
刚买了车,没钱,所以根本谈不上去国外旅游。

注意

强调目前没有时间,金钱或精力去做某事.

89.〜とおり(に)/どおり(に)

接続 動辞書形・た形/名ーの＋とおり(に)

名＋どおり(に)

説明 “按照…””正如…那样”

例文

レシピブックに書いてあるとおりにオムライスを作ってみたが、失敗した。
按照菜谱书上写的试着做了蛋包饭,但是失败了.

お祭りは予定どおりに午後３時から始まった。
庙会按预定计划下午三点开始了。

課長は自分が思ったとおりにならないと、すぐ機嫌が悪くなる。
课长一有不顺心的事,就马上会心情不好。

注意

相当于「〜と同じに」,常接在「思う/言う/想像する/考える」等表示思考和言语的动词后,表示与所想的或所说的内容相同.注意前面直接接名词时只能用「〜とおり」.

90.〜とか

(1)〜とか/とかいう

接続 文の普通形/名＋とか/とかいう

説明 表示不确定,”说是…什么的”

例文

A「ああ、だめだ。明日の試験はきっと受からないよ。」.
B「もう、すぐだめとか言って諦めるんだから。しかっりしなさいよ。」
A:”啊,不行了,明天的考试我肯定过不了的”.
B:”真是的.动不动就说不行什么的,轻易地放弃.你振作一点嘛”.

中村さんとかいう人から電話が入ってきた。
一个叫中村什么的人来过电话.

鈴木さんは今日は病気で仕事を休むとか。
铃木说是今天生病休息请假什么的.

注意

表示把听到的内容传达给别人,但是对传达的内容不是很确定.「〜とかいう」后面可以接名词.

(2)〜とかいうことだ/とかいう話だ

接続 文の普通形＋とかいうことだ/とかいう話だ

説明 表不确定的传闻,”好像听说”

例文

ニュースによるよ今年の冬は例年より寒いとかいうことだ。
好像听说新闻说今年的冬天会比往年冷.

田中さんは結婚したら仕事をやめるとかいう話だ。
据说田中结婚后会辞掉工作。

注意

表示传闻,虽然说话人不是很清楚,但是听说过这么一件事情.

(3)〜とかで

接続 文の普通形＋とかで

説明 表示听到的理由,”说是因为…什么的”

例文

姉が大学を中退してフランスに行きたいとかで、我が家は大騒ぎになっている。
姐姐说想要从大学退学去法国什么的,家里闹翻了.

台風が近づいているとかで、飛行機が欠航になった。
说是因为有台风登陆,飞机停飞了.

注意

表示从别人那里听到某种原因,理由,常用于口语.「とか」表示传闻,「で」表示原因.

85.〜ということだ

接続 文の普通形＋ということだ

説明1 表示得出某个结论,”就是…””也就是说…”

例文1

父はまだ海外出張から帰ってきていない。つまり今日の私にの卒業式には参加できないということだ。
爸爸去国外出差还没有没有回来.也就是说不能参加我的毕业典礼了.

こんな時間になっても橋本くんは姿を現さない。今日も遅刻ということだね。
到这个时间桥本都还没来.就是说他今天又迟到了。

注意1

「こと」在这里表示”事情”.前项提出一个事实,作为充分的依据,由此得出后项的结论.

説明2 表示传闻,”听说…”

例文2

友達の話では、鈴木さんはなんと宝くじで1000万円当たったということだ。
听朋友说,铃木竟然中了1000万日元的彩票.

田中さんは通勤途中で交通事故に巻き込まれたということだ。
听说田中在上班的路上遇到了交通事故。

注意2

相当于「〜そうだ」,直接引用的感觉很强.

86.〜というものではない/というものでもない

接続 文の普通形＋というものではない/というものでもない

説明 表示否定,”并非…”

例文

外国語の勉強は、単語や文法を覚えればいいというものではない。その使い方をちゃんと理解するのが重要である。
学习外语,并不是记住单词和语法就可以了.真正理解其用法是很重要的.

就職先を決める時は、給料が高ければいいというものではなく、本当にやりたい仕事を第一に考えるべきだ。
在决定就职单位的时候,并非只要工资高就好,应该首先考虑自己真正想做的工作是什么.

お金があれば幸せになれるというものでもない。
并非有钱就一定会幸福.

注意

表示说话人认为某个想法或说法并不全面.「〜というものでもない」比「〜というものではない」否定语气要稍微弱一些.

87.〜というより

接続 文の普通形＋というより

　　　ナ形語幹/名＋というより

説明 “与其说…,还不如说…”

例文

母とはとても仲がよく、親子というより年の離れた姉妹のような感じだ。
我和母亲关系非常好,与其说是母女,不如说是年龄差距大的姐妹.

今回の予選敗退の原因は運が悪ったというより実力不足だと言ったほうがいい。
这次预赛失利的原因,与其说是运气不好,不如说是实力不够更恰当。

仕事ができないというより、仕事をしたくないといったほうが適当だと思う。
我觉得与其说是做不了这份工作,不如说是不想做更恰当.

注意

名词和ナ形后面不接「だ」,直接接 「というより」也可以

88.〜といっても

接続 動/ナ形/イ形の普通形＋といっても

　　　名＋といっても

説明 表示转折,”虽说…”

例文

A「先週から、アルバイトを始めたよ」。
B「へえ、頑張ってるね。」.
A「といっても、週に１回４時間なんだけどね」。
A:”我从上周开始打工了”.
B:”是吗,你很努力”。
A:”虽说如此,一周也就一次总共四个小时而已”.

普段は外食だけど、週末は自炊をしている。料理を作るといっても、カレーぐらいの簡単なものしか作れないけど。
平时都是在外面吃,周末会自己做饭.虽说是做饭,也只会做咖喱之类的比较简单的料理.

一口に保険といっても生命保険や医療保険など色々な種類がある。
虽然都统称为保险,但也有人寿保险,医疗保险等很多不同的种类.

注意

①后项可以表示并没有前项所叙述的那样严重,程度并没有那么深(如第一个和第二个例句).
②前项做了简单的归纳,后项表示实际上比前项归纳的要复杂(如第三个例句).
③「いうっても」可以作为接续词单独使用(如第一个例句).

81.〜てならない

接続 動て形＋ならない

　　　イ形て形＋ならない

　　　ナ形語幹＋でならない

説明 “…得不得了””非常…”

例文

急な仕事で、10年ぶりの同窓会に行けなくなり、残念でならない。
因为有紧急的工作,去不了十年的同学会,真的是很遗憾.

気のせいかもしれないが、最近上司に無視されているような気がしてならない。
也许是心理作用吧,最近总觉得好像是被上司忽视了。

大学の時もっと勉強しておけばよかったと悔やんでならない。
要是大学时多学点东西就好了,现在后悔得不得了.

注意

强调程度很高.表示情不自禁地去做某事,自己无法控制.

82.〜てはいられない

接続 動て形＋はいられない

説明 “不能…”

例文

仕事がたまっていて、三連休といっても休んではいられない。
积压了很多工作,虽然说是连续放假三天,也不能休息了.

事の真相を知った以上、もう黙ってはいられない。
既然知道了真相,就不能再沉默下去了.

来週は大事な期末試験なので、のんびりしてはいられない。
下周就是重要的期末考试了,不能再想平时那样悠闲了.

注意

表示迫于某种情况,不能再继续以前的状态了,或急于想做某件事,包含危机感或紧迫感等.

83.〜てばかりはいられない/てばかりもいられない

接続 動て形＋ばかりはいられない/ばかりもいられない

説明 表示不能总是以某种状态继续下去,”(也)不能总是”

例文

どんなに辛くても、泣いてばかりもいられない。もっと強く生きて行かなければならない。
无论多痛苦,也不能总是哭.必须更加坚强地活下去.

体の調子が悪いからといって、このまま会社を休んでばかりはいられない。
虽然说最近身体状况不太好,但不能总是这样请假不去上班。

今日の試験に勝ってからといって喜んでばかりはいられない。来週は決勝戦だ。
虽说今天赢了,但不能总是沉浸在喜悦当中.下周就是决赛了.

注意

常与表示感情的词搭配使用,包含了说话人对现状的不安,”不能总是这么下去,不能沉浸在…当中”

84.〜てはじめて

接続 動て形＋はじめて

説明 “在…之后才…”

例文

外国に住んでみてはじめて日本の治安のよさを実感した。
在国外居住之后才深切地体会到了日本的治安之好.

ビジネスの世界では、お互いの信頼関係があってはじめて商談や取引が成り立つのだ。
在商界,彼此之间建立相互的关系之后,谈判和交易才能达成.

人は病気になってはじめて健康の大切さに気づく。
人生病了之后才会意识到健康的重要性.

別れてはじめて相手のよさに気づく場合もある。
有时在分手之后才发现对方的优点.

注意

经过前项的事情之后,对过去没有在意的事情有了新的认识或感悟;或有了前项作为基础,后项才能得以实现.前项是后项发生的理由或者契机.

79.〜てまたらない

接続 動て形＋たまらない

　　　イ形て形＋たまらない

　　　ナ形語幹＋でたまらない

説明 “…得受不了””非常…”

例文

海外出張中は、幼い子どもたちにあいたくてたまらなかった。
在国外出差的时候,非常想念年幼的孩子们.

みんなの前で、英語の発音の間違いを指摘されて、恥ずかしくてたまらなかった。
在众人面前被指出了英语发音的错误,真是羞愧难当。

蚊に刺された脚が痒くてたまらない。
脚被蚊子叮了,奇痒难忍.

注意

①与「〜てしょうがない」意思大致相同,强调忍受不了某种感觉.不能接在表示自发的动词或词组之后,如「思える/気になる」.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等.

80.〜てでも

接続 動て形＋でも

説明 表示采取强硬的手段,”就算…也要…”

例文

今日は、徹夜してでもこの仕事を終らせなければならない。
今天就算通宵也要把这个工作做完.

姉は昔から「欲しいものは借金をしてでも買う」タイプの人だ。
我姐姐以前开始就是”想要的东西,就算是借钱也要买的”那种人.

家を売ってでも子どもの手術の費用を用意したい。
就算是卖房子,也要凑齐孩子的手术费用.

注意

表示为了达到某一目的而采取强硬的手段,后项表达了说话人强烈的愿望和意志.

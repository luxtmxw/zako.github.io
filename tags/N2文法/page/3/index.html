66.〜たばかり

接続 動た形＋ばかり

説明 表示刚刚做完某事,”刚…”

例文

このゲーム機は昨日買ってきたばかりなのに、もう弟に壊されてしまった。
这个游戏机是昨天刚买来的,却已经被弟弟弄坏了.

A「ねえ、そのマンガ、読み終わったら貸して」。
B「いいけど、まだ読んでないよ。図書館で借りてきたばかりなんだ」。
A:”哎,那本漫画看完借给我看吧”。
B:”可以啊,不过我还没看完,刚从图书馆借的”。

ここに引っ越したばかりですから道もよくわかりません。
因为刚搬到这里,所以路还不是很熟悉.

出張から帰ってきたばかりで、とても疲れている。
刚刚出差回来,很疲惫.

注意

表示某个动作刚发生完,没有多长时间。

67.〜たびに

接続 動辞書形/名ーの＋たびに

説明 “每当…””每次…”

例文

今住んでいるところは幹線道路に近く、大型車が通るたびに揺れる。
现在住的地方靠近主干道,每次有大型车通过,房屋都会晃动.

この曲を聞くたびに、大学時代のことを思い出す。
每当听到这首曲子,就会想起大学时代.

姉は旅行に行くたびに、旅先から絵葉書を送ってくれる。
姐姐每次出去旅行,都会从当地给我寄明信片.

注意

每当发生前项,就总是会发生后项的事情.「この度」是惯用表达,表示”这次”

68.〜だらけ

接続 名＋だらけ

説明 “满是…””都是…”

例文

小さい頃は、兄たちと泥だらけになって遊ぶのが大好きだった。
小时候特别喜欢喝哥哥们一起玩,玩得浑身是泥.

あいつはお金持ちに見えているが、実は借金だらけだそうだ。
那家伙看起来是有钱人,但据说他负债累累.

こんな間違いだらけの文章はとても読むきにならない。
这种错误百出的文章,实在没心情看下去.

注意

表示”到处都是…””充满了…”,前面的名词一般带有负面意义,表示说话人负面的评价.惯用的搭配有「泥だらけ(沾满泥)/傷だらけ(伤痕累累)/間違いだらけ(错误百出)/借金だらけ(负债累累)/血だらけ(到处是血)/しわだらけ(皱巴巴)」等。

63.〜たところ

接続 動た形＋ところ

説明 表示做了前项的动作之后,发生或发现了后项的结果

例文

化粧品売り場で新製品の問い合わせをしたところ、試用品をもらった。
在化妆品专柜询问了下新产品,他们送了我试用装.

近くの交番に聞いたところ、昨日失くした定期券が届いていた。
去附近的派出所问了下,得知昨天丢失的月票被人捡到送过来了.

彼にお願いしたところ、快く引き受けてくれた。
我拜托了他,他很爽快地答应下来了.

注意

相当于「〜たら」。

64.〜たところが

接続 動た形＋ところが

説明 表示逆接,”然而…””虽然…但是…”

例文

田中さんならわかるだろうと思って聞いてみたところが、彼にもわからないという。
我想田中的话大概会知道的,我就问了问,可他说他也不知道.

今朝は早めに出かけたところが、交通事故に巻き込まれ遅刻してしまった。
今天早上提早出门,可遇到了交通事故,迟到了.

昔住んでいた町を訪ねたところが、様子がすっかり変わっていて迷ってしまった。
去了曾经住过的街道,完全变了样,都不认识了.

注意

表示逆接时,一定要用「〜たところが」的形式.后项的结果是与预想,期待相反.

65.〜たとたん(に)

接続 動た形＋とたん(に)

説明 “一…就…””刚…就…”

例文

改札口を出たとたんに、傘を電車の中に忘れたことに気づいた。
刚出检票口,就发现把伞忘在电车里了.

三日ぶりに会った幼い娘は、私の顔を見たとたん、わあーと泣き出した。
时隔三天,年幼的女儿一看到我,就”哇”的一声哭了起来.

火災報知器のベルが鳴ったとたんに、みんな慌てて外に飛び出した。
火灾报警器一响,人们就慌张往外面跑了出去.

注意

①前项刚发生,接着就发生了后项,后项多是意料之外的事情,并且是说话人亲历的已经发生的事情了,因此句末只能是过去时.
②后项不能是表示命令,意志,推量,否定的内容.

61.〜だけあって/だけに/だけのことはある/

接続1 動/ナ形/イ形の名詞修飾形＋だけあって/だけに/だけのことはある

名＋だけあって/だけに/だけのことはある

説明1 “与…相称””不愧是…””到底没白…”

例文1

鈴木くんは北国で育っただけあって、寒さに強い。
铃木不愧是在北方长大的,不怕冷.

山田先生は10年も教壇に立っているだけあって、教え方がうまく生徒たちに人気だ。
山田老师不愧是有着十年的教学经验,擅长教学,很受学生欢迎.

彼女は英語の発音がとてもきれいで、アメリカへ留学しただけのことはある。
她的英语发音很标准,到底没有白去美国留学.

注意1

表示与其身份,地位,经历等相称,后项对其进行正面的评价.此时,「だけ/だけあって/だけに/だけのことはある」等几种形式都能使用.常与「さすがに」呼应使用.

接続2 動/ナ形/イ形の名詞修飾形＋だけに

名＋だけに

説明2 “正因为…更加…”

例文2

山を登るのに苦労しただけに、頂上で見た風景は最高だった。
正因为登山的过程很辛苦,爬到山顶后看到的风景才觉得格外美丽.

妹はもともと体が弱いだけに、風邪を引いても心配だ。
正因为妹妹身体本来就很虚弱,所以就算是感冒也让人担心.

こちらは壊れやすいものだけに、取り扱いには十分気をつけてください。
这个物品容易损坏,所以请轻拿轻放.

注意2

相当于「〜だけに、なおさら」,表示”正因为前项的原因,后项才更加…”,此时一般使用「だけに」。

接続3 動/ナ形/イ形の名詞修飾形＋だけに

名＋だけに

説明3 “正因为…反倒…”

例文3

体に自信があるだけに、かえって無理をして体を壊してしまった。
正因为对身体有信心,所以反倒逞强弄坏了身体.

優勝することを期待していただけに、負けたときの失望は大きかった。
正因为一直很期待取得胜利,所以失败时非常失望.

鈴木課長は普段静かなだけに、怒り出すとすごく怖い。
正因为铃木课长平时很温和,生起气来十分可怕.

注意3

相当于「〜だけに、かえって」,表示”正因为…反而更加…”,此时一般使用「だけに」。

62.たとえ〜ても

接続 たとえ＋動て形＋も

　　 たとえ＋イ形て形＋も

　　 たとえ＋ナ形語幹/名＋でも

説明 表示让步,”即使…也…”

例文

たとえ失敗しても、努力したことは決して無駄にならないと思います。
我觉得即使失败,所做的努力也绝不会白费.

たとえ両親に賛成されて結婚しても、一生うまくいくという保証はない。
即使得到了父母的赞同结婚了,也无法保证一辈子都会幸福.

たとえ苦しくても途中であきらめることなく、最後まで頑張るつもりだ。
即使很辛苦,也不会中途放弃,打算坚持到最后.

注意

后项除了与「〜ても」搭配之外,还可以与「〜たところで/としても」等搭配使用.

59.〜せいで/せいか/せいだ/せいにする

接続 動/ナ形/イ形/名の名修飾形＋せいで/せいか/せいだ/せいにする

説明 表示原因,”因为…”

例文

今年は二月の気温が低かったせいか、桜の開花が例年より遅い。
今年可能是因为二月的气温比较低,樱花的花期比往年晚一些.

ストレスのせいか、最近夜ぐっすり眠れない日が多い。
也许是因为压力大,最近经常晚上睡不好.

お腹が痛いのはきっと賞味期限が切れた牛乳を飲んだせいだ。
肚子痛肯定是因为喝了过期的牛奶.

自分のミスを人のせいにするのはよくないよ。
将自己的过错归咎于别人是不对的.

注意

①表示由于某个原因导致了不好的,消极的结果
②「〜せいで」用于句中.「〜せいだ」用于句末,常用于句型「〜のは〜せいにだ」。「〜せいか」表示不确定的原因,”可能是因为…”
③句型「〜せいにする」表示”归咎于…””怪罪于(某人)…”

60.〜たいばかりに/ほしいばかりに

接続 動ます形＋たいばかりに

　　 名ーが＋ほしいばかりに

説明 表示原因,”就是因为想”

例文

ブランドバッグを手に入れたいばかりに、夏休みは一生懸命アルバイトをした。
就是因为想买名牌包,暑假才拼命打工的.

好きな歌手のコンサートを見に行きたいばかりに、病気にかこつけて休みを取った。
就是因为想去看喜欢的歌手的演唱会,才会生病为借口请假.

仕事を早く終わらせたいばかりに、昼休みも取らずに頑張った。
就是因为想早点把工作做完,中午不休息努力干活.

注意

表示为了实现某种急切的愿望而不辞辛苦去做某事,或者不择手段去做某事,包含”一心想…”的意思.

57.〜ずにすむ/ないですむ/なくてすむ

接続 動ない形＋ずにすむ

動ない形＋ないですむ

動ない形＋なくてすむ

する→せずにすむ/しないですむ

説明 “不…(也可以解决)””不用…就可以了”

例文

帰国する先輩から自転車を譲ってもらったので、新しいのを買わなくてすんだ。
即将回国的前辈把自行车转让给了我,所以我就不用买新的了.

今朝乗っていた電車が一時運転見合わせになったけど、早めに家を出たので遅刻せずにすんだ。
今天早上乘坐的电车停止运行了一段时间,但是因为提早出门了,所以没有迟到.

今回は風の症状が軽かったので、病院に行かずに済んだ。
这次的感冒症状很轻,所以就不用去医院了.

注意

可以表示”不必做原来预定要做的事情”(如第一个例句和第三个例句),也可以表示”避免了可能会发生的事情”(如第二个例句)

58.〜ずにはいられない/ないではいられない

接続 動ない形＋ずにはいられない

　　 動ない形＋ないではいられない

　　 する→せずにはいられない/しないではいられない

説明 表示迫切的心情,”不由得…””不得不…””不能不”

例文

女性は三十歳を過ぎると、どうも年齢をきにせずにはいらなくなる。
女性过了30岁,就不得不在意起自己的年龄了.

2時間も待たされて、担当スタッフに文句を言わずにはいらなかった。
让我等了两个小时,所以忍不住队对责人抱怨了几句.

彼は自分のことをほうっておいてでも苦境にいる友達を助けないではいられない人だ。
他这个人,即使不管自己的事情也忍不住要去帮助处于困境的朋友.

注意

表示控制不住做某事的心情,如果不做某事,心里会很不舒服

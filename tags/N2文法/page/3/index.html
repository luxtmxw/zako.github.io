24.〜かと思えば

接続1　動辞書形・た形/名＋かと思えば

説明1　“原以为…却…”

例文1

九月に入ってやっと涼しくなったかと思えば、また急に暑くなったりして残暑が厳しい。
到了9月,原以为终于要凉快了,却突然变热,秋老虎还很厉害.

息子は学校から帰ってきたかと思えば、またすぐ遊びに出かけていった。
儿子从学校回来了,却又马上出去完了.

注意1

后项是与说话人预想相反的事情

接続2 動辞書形+かと思えば

説明2 表示并列,”既…又…”

例文2

老後を一人で生きることを選ぶ熟年夫婦があるかと思えば、新しい旅立ちに向けて、退職後豪華な旅行にでる夫婦もある。
有的中老年夫妻选择独自一人度过晚年,也有些夫妻打算一同踏上新的旅程—退休后参加豪华旅游.

百歳を越えて生きる人がいるかと思えば、一歳になることもなく、この世を去る赤ん坊もいる。
既有活过百岁的寿星,也有还不到一岁就离开人世的婴儿.

注意2

常与「ある/いる」搭配使用,构成「〜かと思えば〜もある/いる」表示”既有…又有…”.

25.〜か〜ないかのうちに

接続　動辞書形＋か＋動ない形ーない＋かのうちに

説明 表示两件事情几乎同时发生,”刚…就…”,”还没…就…”

例文

太郎は「ご馳走様」と言い終わるか終わらないかのうちに、席を立って自分の部屋に入った。
太郎刚说完”我吃饱了”,就离开位子回到自己的房间了.

息子は夕飯を食べ終わるか終わらないかのうちに、またゲームを始めた。
儿子刚吃完晚饭,就开始打游戏了.

電車に乗るか乗らないかのうちに、ドアがしまってしっまた。
刚乘上电车,门就关了.

注意

①使用同一个动词的肯定和否定形
②前后项的两个动作几乎是同时发生,并且包含了”前项动作甚至还没有完成就发生了后项的动作”的意思.
③用于描述已经发生的事实,后项不能是表示命令,意志,推量,否定的内容,并且只能是过去时.

26.〜か何か

接続　動普通形/名＋か何か

説明 表示泛指,”…什么的”

例文

来週の日曜日は「母の日」なので、母にカーネーションの花束か何かを送ろうと思う。
下周就是母亲节了,我想送妈妈一束康乃馨什么的.

仕事の合間、お茶か何かを飲む。
工作间歇,喝喝茶什么的.

パソコンはウィルスに感染したか何かでフリーズして動かなくなった。
电脑因为中毒什么的死机了.

注意

前项内容并不是特定的.「〜か何か」也可以用「名＋や何か」的形式,表示”…或者…(之类的)”.另外,根据「何」后面具体所用到的动词,可将「何」后面的助词「か」变成「を」或「で」等.

A:「ねえ、見て。この本、ベストセラーだって、買っちゃったよ。」
B:「ああ、それって、最近テレビや何かで話題になっている小説でしょ。」
A:你看,听说这本书是畅销书,我就买了.
B:啊,那本书啊,就是最近在电视呀什么的上面经常被提到的那本小说吧.

23.〜(か)と思うと/(か)思ったら

接続1　動た形＋(か)と思うと/(か)思ったら

説明1　表示事情几乎同时发生,”刚…就…”

例文1

朝、息子が慌てて学校に行ったかと思ったら、すぐ「ケータイ！」と叫びながら戻ってきた。
儿子早上刚匆匆忙忙地出门上学,就边喊着”我的手机”边返回了家里.

急に空が暗くなったかと思うと、激しく雨が降り始めた。
天空刚一暗下来,马上就开始下大雨了.

注意1

① 表示前项的事情发生后,紧接着发生了后项的事情,并且后项多为意料之外的事情,包含说话人惊讶的心情.
② 只能用于描述客观事实或他人的情况,不能用于说话人自身的行为。
③ 用于描述现实的状况,后项不能是表示命令,意志,否定的内容.

接続2 動辞書形＋(か)と思うと/(か)思ったら

説明2 表示结果与说话人预想的相反,”原以为…却…”

例文2

やっと部屋がきれいに片付いたと思ったら、子どもたちが学校から帰ってきてまた散らかし始めた。
原以为终于把房间收拾干净了,可结果孩子们从学校回来之后又开始弄得乱七八糟的.

息子は机に向かって勉強を始めたかと思うと、携帯電話でゲームをやり始めた。
我原以为儿子坐在桌前开始学习了,可结果他却玩起了手机游戏.

やっと仕事が終わったかと思ったら、また新しい仕事を頼まれた。
原本以为工作终于结束了,结果又安排了新的工作.

注意2

后项表示与说话人预想相反的事情.

21.〜がちだ/がちの/がちな

接続　動ます形/名＋がちだ/がちの/がちな

説明　表示倾向,”容易……”,”往往会”

例文

この野菜ジュースは現代人に不足しがちな栄養素を補ってくれる。
这个蔬菜汁可以补充现代人容易缺乏的营养元素.

冬は、部屋の換気を忘れがちだから、気をつけよう。
冬天经常会忘记给房间通风换气,所以要多加注意.

この時期になると、風邪などで学校を休みがちな生徒が増える。
一到这个时期,因为感冒等而经常请假不来上课的学生会增多.

注意

① 表示容易发生某种负面的,消极的事情.
② 「がち」的活用相当于ナ形容词.结句用「〜がちな」;修饰名词用「〜がちな/がちの」。
③ 「がち」前面接名词时表示容易产生该名词所表达的状态;接动词时,表示频繁发生某种行为,强调发生次数多.

22.〜かというと/かといえば

接続1　動普通形/イ形＋(の)かというと/かといえば

ナ形語幹/名＋(なの)かというと/かといえば

説明1　“至于是否…”,”是不是…就…”

例文1

一流大学を卒業したから一流会社に就職できるかというと、そんなことはない。
是不是一流大学毕业就能进入一流公司工作,事实并非如此.

昨日携帯電話を忘れて出かけだ、困ったことがあったかといえば、それでもなかった。
昨天出门忘记带手机了,要说有没有很不方便,其实也还好.

歌を聞くのが好きだが、歌うのが上手なのかといえば、それでもない。
虽然喜欢听歌,但是不是就很会唱歌,其实也并非如此.

姉はどちらかというと、母の方に似ている。
总的来说,姐姐像母亲.

注意1

①前项提出一个事实或前提,由此自然而然地推导出一个结果,可实际上结果于此相反,「かというと」和「かといえば」可以替换.句末是否定的形式,常用的有「それでもない/そうではない/そうとは限らない/そういうわけでもない/そこまでは行かない」等.
②「どちらかというと/どちらかといえば」是固定用法,表示”说起来,总的来说……”

接続2 疑問詞＋かというと/かといえば

説明2 提出疑问后,给出原因或解释,”要说…那是因为…”

例文2

夏休みが嫌いだ、なぜかというと、宿題が多くてあまり遊ぶ時間がないからだ。
我不喜欢暑假.要说为什么,那是因为作业太多,没什么时间玩.

どうしてこんな所に物を置いたかというと、後で運ぶ時に便利だからだ。
说到为什么要把东西放在这里,那是因为以后搬得的时候比较方便.

注意2

前项提出问题,后项给出解答,可用于自问自答的场合.「かというと」和 「かといえば」可以替换.后项与「〜からだ/のだ/ためだ」呼应使用,疑问词可以直接接「かというと」,也可以加上一个小句后再接「かというと」。如第一个例句中的「なぜかというと」也可以说成「なぜ夏休みが嫌いかというと」。

20.〜がたい

接続　動ます形＋がたい

説明　“难以……”

例文

明日、留学を終えて国に帰る。これまで仲良くしていた友だちとは本当に別れがいた。
明天就要结束留学生活回国了.要与好友们告别,很是不舍.

お年寄りを騙してお金をとる犯罪は許しがたい。
骗取老年人钱财的犯罪行为是不能饶恕的.

あの正直な彼が嘘をつくとは、信じがたい。
那么诚实的他居然会撒谎,真实难以置信.

注意

①是书面语,常与表示认知和说话有关的动词搭配使用,如「理解しがたい(难以理解)/想像したい(难以想象)/賛成しがたい(不能赞成)/信じがたい(难以置信)/言いがたい(难以启齿)/表がたい(难以表达)/許しがたい(不能饶恕)/受け入れがたい(难以接受)/認めがたい(不能承让)」.
②「動かしがたい事実」为惯用表达,表示”不能否认的事实”

18.〜が契機で/が契機になって/を契機に(して)/を契機として

接続　動普通形ーの/名＋が契機で/が契機になって/を契機に(して)/を契機として/

説明　“以……为契机,机会”

例文

来月子供が生まれる。これを契機に、禁煙を決意した。
下个月宝宝要出生了,以此为契机,我下定决心戒烟.

会社はロボット開発を契機に、新しい事業を展開した。
公司以开发机器人为契机,开展了新的业务.

今回の業務提携を契機に、今後さまざまな分野での交流が期待される。
以这次的业务合作为契机,期待今后在多个领域进行交流.

注意

与包含「きっかけ」的几个句型意思相同,是其书面语.

19.〜かけだ/かけの/かける

接続　動ます形＋かけだ/かけの/かける

説明　表示某动作开始后还没有结束,”没…完”,”…做到一半”

例文

昨日買ってきたイチゴが、もう腐りかけている。
昨天买来的草莓已经开始烂了.

食べかけのケーキにラップをして冷蔵庫に入れた。
把吃了一半的蛋糕用保鲜膜包起来,放进了冰箱.

家の前で死にかけていた野良猫を拾って動物病院に連れてった。
在家门口捡到一只块死的野猫,带它去了动物医院.

注意

①既可以表示具体的动作做到一半,没有结束(如第二个例句),也可以表示某个状态已经开始发生(如第一第三个例句).
②「〜かけだ」用于句末结局,「〜かけの」后面接名词,「〜かける」可用在居中修饰名词,也用在句末.

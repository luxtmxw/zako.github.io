51.〜際(は)/際に(は)

接続　動辞書形・た形/名ーの＋際(は)/際に(に)

説明　“在…时候””借…时机”

例文

面接が終わって退室する際は、会釈をし、静かにドアを閉めましょう。
面试完离开房间的时候,要点头致意,再轻轻地关上门.

ご乗車の際は、乗務員に乗車券をご提示ください。
乘车的时候,请向乘务员出示您的车票.

先月帰省した際に、高校の先生を訪ねた。
上个月回老家时,借机去拜访了高中的老师.

注意

语气比「〜とき」更书面,并且带有”时机,契机”的意思.

52.〜最中(に)/最中だ

接続 動ている形/イ形/名ーの＋最中(に)/最中だ

説明 表示某动作正在进行,”正在…”

例文

休みの日に、外食をしている最中に、会社から予定外の仕事の連絡が入った。
休息天在外面吃饭的时候,接到公司的通知,说是临时有工作需要处理.

商談の最中だから、携帯電話をサイレントモードに設定しておいた。
现在正在谈判,事先把手机设定成了静音模式.

あの件に関しては、今、部内で検討している最中だ。
关于那件事,现在正在内部讨论中.

原稿の締め切りで忙しい最中に、電話をかけてこないでください。
现在正忙着赶交稿日期,你不要打电话过来了.

注意

表示正在做某事时,发生了后项的事情,后项事情多带有突发性和偶然性.多接在动词的持续体和「名ーの」的后面.

53.〜ざるを得ない

接続 動ない形＋ざるを得ない

　　　するー>せざるを得ない

説明 “不得不”

例文

大雨で、サッカーの試合は中止せざるを得なかった。
由于大雨,这场足球比赛不得不终止.

勇気は認めるが、こんな登山計画は無謀だと言わざるを得ない。
虽然你勇气可嘉,但不得不说这个登山计划欠斟酌.

明日から出張の予定だったんだけど、インフルエンザにかかってしまい、休みざるを得ない。
原本计划明天出差的,但是得了流感,不得不请假休息.

注意

属于书面语,表示除此之外,别无他法.包含”本来不想做,但是迫于某种压力或原因而不得不去做”的意思

49.〜ことは〜が

接続　動/イ形の名詞修飾形＋ことは＋動/イ形の普通形＋が

ナ形語幹ーな＋ことは＋ナ形語幹＋だ＋が

説明　表示让步,”…是…,但是…”

例文

あの小説は読んだことは読んだが、方言がたくさん出てきて、話の筋があまりよくわからなかった。
那部小说看是看过了,但是因为有很多方言,所以故事情节没怎么看明白.

このレストランはおいしいことはおいしいが、値段が高すぎる。
这家店好吃是好吃,就是太贵了.

スペイン語はわかることはわかるが、話し方が早いとうまく理解できない。
西班牙语懂是懂一点,但如果说得快的话就无法很好地理解.

注意

①表示虽然承认前项,但并不能给予肯定,后项提出消极的方面。
②前后项是使用同一个动词或同一个イ形、ナ形

50.〜ことはない

接続 動辞書形＋ことはない

説明 “没必要…””用不着…”

例文

今回の失敗はあなたのせいじゃないんだから、謝ることはないよ。
这次的失败不是你的错,所以没有必要道歉.

バーベキューに必要な道具は、現地でレンタルできるので、わざわざ持っていくことはない。
烧烤需要的器具,可以现场租借,所以没有必要特意带过去.

資料に詳しく書いてあるから、いちいち説明することはない。
资料上面都详细写着,就不用一一解释了.

注意

相当于「〜する必要はない」,用于说话人对对方的忠告或建议等.与「何も/わざわざ」呼应使用,表示”并没有必要…””没有必要特意…”.

45.〜ことから

接続　動/イ形の名詞修飾形＋ことから

ナ形語幹＋な/である＋ことから

名＋である＋ことから

説明　表示原因或依据.”因为…””从…来看”.

例文

彼はバレーボールのルールをよく知っていることから、練習試合の審判員に選ばれた。
因为他熟知排球的规则,所以被选为练习赛的裁判.

この橋は、姿がめがねのように見えることから、「めがね橋」と呼ばれるようになった。
因为这座桥的外形看起来很像眼镜,所以被称之为”眼镜桥”.

指紋が一致たことから、彼は犯人に特定された。
根据指纹相吻合的情况,他被认定为罪犯.

注意

可以表示前项是后项的原因(如第一和第二个例句),还可以表示前项是后项的依据(如第三个例句)。

46.〜ことだ

接続 動辞書形・ない形ーない＋ことだ

説明 表示忠告或建议等,”应该…””不要…”

例文

友たちをたくさんつくりたかったら、自分から積極的に話しかけることだ。
想多交朋友的话,就要自己主动地找别人说话.

健康的に痩せたかったら、食事の量を控え、適度な運動をすることだ。
想要健康的瘦身的话,就要控制食量,适度运动.

読解力をつけたいのなら、毎日、新聞を読むことだ。
想要提高阅读能力的话,就应该每天看报纸.

注意

用于说话人对对方的忠告,提醒等,建议对方这样做会更好.

47.〜ことだから

接続　名＋の＋ことだから

説明　表示说话人判断的依据,”因为是…”

例文

小林さんは親切な人で、よく後輩の面倒を見る。そんな彼のことだから、頼めばきっと助けてくれるよ。
小林为人热情,经常照顾晚辈.因为他是这样的一个人,所以你去拜托他的话,他肯定会帮忙.

時間にルーズな彼のことだから、今日も遅れてくるだろう。
时间观念淡漠的他,今天也会迟到的吧.

品質管理に厳しい田中さんのことだから、そんなミスはしないでしょう。
田中对质量监管要求很严格,想必不会犯那种错误吧.

注意

一般接在表示人物的名词后面,表示该人物具有某种特征,说话人通过对他的这一性格特征的了解作出某种判断,包含”因为他是这样的一个人,所以…”的意思.

48.〜ことなく/こともなく

接続　動辞書形＋ことなく/こともなく

説明　表示否定,”不…”

例文

どんな時でも落ち込むことなく、強く前向きに生きられる人になりたい。
我想成为无论何时都不气馁,能坚强乐观地生活的人.

今日は部内検討会だから、新人でも遠慮することなく、自分の意見を言っていい。
因为今天是部门内部讨论会,所以新人也不用顾虑,可以发表自己的意见.

彼は一日も休むことなく小説を書き続けている。
他不停的写小说,一天也没有停过.

注意

相当于「〜ないで/ずに」,属于书面语.「〜こともなく」比「〜ことなく」语气更强.

43.〜くらい/ぐらい

(1)〜くらい/ぐらい

接続　動辞書形・ない形ーない/ナ形語幹ーな/イ形/名＋くらい/ぐらい

説明　表示程度.

例文

スロージョギングは、普段歩くくらいのスピードで行うジョギングである。
慢跑就是指平时走路的速度来跑步.

A「このお店のロールケーキ、おいしいね。」。
B「うん、毎日食べたいくらい好きだよ。」/
A:”这家店的瑞士卷真好吃呀.”
B:”嗯,我很喜欢,几乎每天都想吃”

朝から腰が痛くて、立っていられないぐらいだ。
早上开始腰痛,不能久站.

この1か月は仕事が忙しくて、ジムに行く暇もないくらいだ。
这个月工作很忙,连去健身房的时间也没有.

あいさつぐらいの簡単なフランス語なら私もできる。
如果是寒暄之类的简单的法语,我也会.

注意

前三个例句表示动作和状态的程度,可以与「ほど」替换。第四和第五个例句表示最低程度时,不能用「ほど」,只能用「くらい/ぐらい」。另外,「ほど」表示最高程度时,不能用「くらい」,如「死ねほど忙しい(忙得要死)」不能说成「死ねくらい忙しい」。

(2)〜くらい/ぐらい(〜はない)

接続 動辞書形/名＋くらい/ぐらい(〜はない)

説明 “没有比…更…””…最…”

例文

やりたくないことを無理にやるくらいつらいことはない。
勉强自己去做自己不想做的事情是最痛苦的.

徹夜くらい健康に悪いことはない。
没有比熬夜更损害健康了.

毎日同じ作業を繰り返すくらいつまないことはない。
没有什么比每天反复做同样的工作更无聊了.

注意

此时,「くらい」,提示的事物是最高程度,可以用「ほど」替换.

44.〜ことか

接続　動/ナ形/イ形/名の名詞修飾形＋ことか

説明　“多么…啊”

例文

中学生の息子がまた制服を脱ぎっぱなしにしている。もう何度注意したこたか。
上中学的儿子又把校服脱了乱扔.我都提醒过他多少次了啊.

先週、空港で偶然好きなタレントに会えて、どれほど嬉しかったことか。
上周在机场偶然见到了我喜欢的艺人,别提有多高兴了.

家の近くに地下鉄が通って、どんなに便利なことか。
家附近通了地铁,多么方便啊.

注意

表示非常强烈的感情,前面常与「どれほど/どんなに/なんと/何度」等次呼应使用,如「何度注意したことか(我提醒你多少次了啊)」.

41.〜きる/きれる/きれない

接続　動ます形＋きる/きれる/きれない

説明1　表示做(不)完某个动作,”…(不)完”

例文1

初めて市民マラソン大会に参加したが、頑張って走りきった。
虽然是第一次参加马拉松大赛,但是努力跑完了全程.

この製品はとても人気があるらしく、発売日売りきれてしまったそうだ。
这个产品似乎很有人气,发售当天就卖光了.

こんなにたくさんの料理は、二人では食べきれない。
这么多的菜肴,两个人是吃不完的.

注意1

「〜きる」表示”(做)完”;「〜きれる」表示”能(做)完”;「〜きれない」表示”(做)不完”。

説明2 表示程度深

例文2

スタッフに入会の手続きについて説明してもらったが、納得しきれないところがある。
工作人员向我们说明了入会的手续,但是有些地方我还没有完全理解.

連日の残業で疲れきってしまった。
连续几天加班,累坏了.

彼女は困りきった顔で、呆然と坐っていた。
她一副非常为难的样子,呆呆地坐在那里.

注意2

①「言い切る(断言)/思い切る(下决心)/わかり切る(很明白)」等,一般作为固定的动词来用
②惯用表达有「死んでも死にきれない(死也不甘心)/数えきれないほど(不计其数的)」.

42.〜くせに/くせして

接続　動/ナ形/イ形/名の名詞修飾形＋くせに/くせして

説明　表示转折,”…却…””…可是…””明明…”

例文

夫は子育てに全然協力しないくせに、私のやり方に不満があるようだ。
丈夫完全不帮着带小孩,却好像对我带孩子的方式有所不满.

娘は今日から早起きすると言ったくせに、７時になっても起きてこない。
女儿明明说了从今天开始要早起,但是到了七点了还没起床.

あの人は新入社員のくせして生意気で自己主張が強い。
那个人明明是个职场新人,却很傲慢,很喜欢坚持自己的观点.

注意

表示后项与前项不符合.接在表示人的名词后面时,表示后项与该人的身份不符,带有指责,不满,抱怨的语气.

6.〜上で(は)

接続1　動辞書形/名ーの＋上で(は)

説明1　“在……上”,”在……方面”

例文1

あの二人は戸籍の上では夫婦だが、何年も別居していて他人同然になっている。
那两人虽然在户籍上是夫妻,但是已经分居多年形同陌路了.

暦の上では立秋を過ぎたが、秋の気配がぜんぜん感じられない。
在日历上已经过了立秋,但还丝毫感觉不到秋天的气息.

外国語を勉強する上で単語を覚えることはとても重要だ。
在学习外语方面,记单词很重要.

注意1

前面接名词时,常接「暦/法律/地図/データ(数据)」等.

接続2　動た形/名ーの＋上(で)

説明2　“在……之后”

例文2

入場券の当日販売はありませんので、予めネットで入場券をお買い求めの上、ご入場ください。
当天现场不售门票,所以请您事先在网上购票之后再入场.

添付ファイルを確認した上で、メールの送信を行う。
确认好添加附件之后,再发送邮件.

わたし一人では決めかねますので、社内で検討したうえで、お返事します。
我一个人很难做出决定,所以在公司内部讨论之后,我再给您答复.

実物を見たうえで買うかどうか決める。
看了实物之后再决定买不买.

注意2

有时可以用「名＋の＋上」,后面不用「で」,多用于通知等书面语(如第一个例句).

7.〜上は

接続　動辞書形.た形＋上は

説明　前项表示既定的事实.”既然…就…”

例文

この仕事を任された上は、業務を遂行し実績を上げなければならない。
既然被委任了这项工作,就必须完成任务,做出成绩.

目標を決めた上は、それに向かって前向きに行動しなければならない。
既然确定了目标,就必须朝着这个目标积极地行动.

こうなった上は、覚悟を決めて最後まで頑張るしかない。
既然事已至此,就只好做好思想准备,坚持努力到最后了.

注意

与「〜以上」的用法相同,”既然…”的语气比「〜以上」更强烈.

3.〜以上(は)

接続　動/ナ形/イ形の名詞修飾形＋以上(は)

ナ形語幹/名＋である＋以上(は)

説明　前项表示确定的事实,”既然…就…”

例文

京都に来た以上、絶対金閣寺を見に行ったほうがいいよ。
既然来个京都,就最好去金阁寺看看.

みんなの前で一人でやると宣言した以上は、誰かに手伝ってもらうことはできない。
既然在大家面前信誓旦旦地说要一个人做,就不能请别人帮忙.

学生である以上、学業を第一に考えるべきだ。
既然是学生,就应该把学业放在第一位.

注意

后项常接表示决心,义务,建议,推量等表达说话人意志的内容.

4.〜一方/一方で(は)

接続　動/ナ形/イ形の名詞修飾形＋一方/一方で(は)

ナ形語幹/名＋である＋一方/一方で(は)

説明　“一方面…另一方面…”,”一边…一边…”

例文

労働力が不足している現象について、女性をフルに活用すべきだと主張する人がいる。一方で、女性の社会進出は少子化の原因たと考えている人もいる。
关于劳动力不足的问题,有人主张应该充分利用女性劳动力.而另一方面也有人认为女性进入社会工作是导致少子化的原因.

科学の発展は便利な生活をもたらす一方で深刻な環境汚染を引き起こした。
科学的发展一方面给生活带来了便利,另一方面又引起了环境污染.

彼女は歌手として活躍する一方で、女優としても。頭角を現した。
她一方面活跃与歌坛,另一方面作为演员也崭露了头角.

注意

①前后项可以是相反的,对比的关系(如第一和第二个例句),也可以是并列的关系(如第三个例句).
②有接续词的功能,可用于连接两个句子(如第一个例句)

5.〜一方だ

接続　動辞書形＋一方だ

説明　“不断…”,”越来越…”

例文

去年の日本のGDP成長率は予想を上回った。それに今年度の第一四半期も連続で改善したので、景気回復への期待は高まる一方だ。
去年的日本GDP增长超出了预期.而且本年度第一季度也是持续增长的,所以人们对经济复苏的预期越来越高.

最近の調査によれば、未成年の犯罪率は高くなる一方だ。
根据最近的调查显示,未成年人的犯罪率越来越高.

ここ数年、経済回復のきざしが見えず、就職状況は厳しくなる一方だ。
这几年,经济没有复苏的迹象,就业形势也越来越严峻.

注意

表示某种倾向,状态等朝一个方向不断发展.

1.~あげく(に)

接続　動た形/名ーの＋あげく(に)

説明　表示结果,”最后的……”

例文

町でお祭りがあって人ごみに巻き込まれたあげく、道に迷ってしまった。
镇上有庙会,我被卷入拥挤的人群,最后迷路了.

進路についてさんざん悩んだあげく、大学院への進路を諦めることにした。
关于毕业后的去向,我烦恼了许久,最后决定放弃读研.

兄に農作業を手伝わされたあげく、仕事が遅いなどと文句を言われた。
被迫帮哥哥干农活,最后却被他抱怨说干活慢.

注意

带有消极意义.除了「〜あげく」的形式之外,还有「〜あげくのはてに」的形式,与「〜あげく」的意思相同,但是后项的结果比「〜あげく」更为严重,可用于连接两个句子,有接续词的功能.

2.〜あまり(に)

接続　動辞書形/ナ形語幹ーな/名ーの＋あまり(に)

説明　表示由于程度过甚,由此导致某种结果.”由于过度…而…”

例文

彼女は人目を気にするあまりに、嫌なことも嫌だと素直に言えない。
因为她太在意别人的眼光,即使是厌恶的事情也无法坦诚地说出来.

大好きなアイドルにやっと会えた幸子は、興奮のあまり、気を失った。
幸子终于见到了自己非常喜欢的偶像,由于兴奋过度,晕倒了.

念願の大学に合格し、うれしさのあまりに、飛び上がりそうになった。
考上了向往已久的大学,高兴得快要跳起来了.

注意

常与表达状态或感情的词一起使用,如「忙しさ」「驚き」「うれしい」「喜び」「悲しみ」「感激」「心配」等.

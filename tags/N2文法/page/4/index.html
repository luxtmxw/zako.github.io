31.〜からいうと/からいえば/からいったら/からいって

接続　名＋からいうと/からいえば/からいったら/からいって

説明　表示评论的立场,”从…来说”

例文

A:「今の若者に一番伝えないことは何でしょうか。」
B:「そうですね。教育者の立場からいうと」、自分らしさを大切に、本当にやりたいことを見つけてほしいですね.
A:”您最想对现在的年轻人说的话是什么?”
B:”嗯,从教育的立场来说,我希望年轻人能保持自己的个性,找到自己真正想要做的事情.”

私の経験からいうと、目の前の仕事にベストを尽くすことが大事だ。
以我的经验来说,尽全力去做眼前的工作是很重要的.

彼女の性格からいうと、この仕事には向いていない。
从性格来说,她不适合这份工作.

注意

表示”站在某个立场来说,来判断”,注意前面不能直接接表示人物的名词.如第三个例句中的「彼女の性格からいうと」不能说成「彼女からいうと」。

32.〜からこそ

接続　文の普通形＋からこそ

説明　强调原因,”正因为…”.

例文

いつでもどこでも読める電子書籍がはやっている時代だからこそ、図書館でじっくり本を読む時間は貴重だ。
当今时代,无论何时何地都能阅读的电子书籍十分流行,正因为如此,坐在图书馆里沉下心阅读的时间是宝贵的.

息子に生活費を自分で稼がせるのは、彼のためだと思っているからこそだ。
让儿子自己赚生活费,正是为他着想.

本当にこの仕事が好きだからこそ、30年も続けてこられたと思います。
我觉得正是因为真的喜欢这份工作,所以才能坚持做了30年.

逆境にあったからこそ、いろいろなことを学び成長できた。
正因为身处逆境,才更能学到东西,得到成长.

注意

①「こそ」表示强调,接在表示原因的「から」后面用来强调这个原因很重要,包含”不是因为别的,就是因为这个原因”的意思,是说话人主观的判断,因此不能用于表示客观地原因.
②一般不用于表示负面的原因.

33.〜からして

接続　名＋からして

説明　表示判断的依据,”单从…来看”.

例文

ここはさすがに老舗に温泉旅館ですね。木造の建物からして、長い歴史が感じられる。
这里不愧是老字号的温泉旅馆.单从木制的建筑物来看,就能感受到其悠久的历史.

村田先生の新作小説は、そのタイトルからしておもしろい。
田村老师新的小说,单从书名来看就很有意思.

このレストラン、雰囲気からして結構高そうだね。
这家餐厅,单从氛围来看就好像很贵.

注意

表示某一事物包含多个因素,仅从其中一个因素去判断整体,包含”别的怎么样先不说,单从这一点看就…”的意思

29.〜かのようだ/かのような/かのように

接続　動普通形＋かのようだ/かのような/かのように

説明　“就像…一样”

例文

夕方の空は、ひが燃えているかのような夕日が一面に広がっていた。
傍晚时分,烈火般的夕阳染红了天空.

あの少女の彫刻はとてもよく作られていて、まるで生きているかのようだ。
那座少女的雕像做得很逼真,就像活的一样.

大きな爆発でビルが揺れ、まるで地震が起こったかのようだった。
大楼因为巨大的爆炸而晃动,简直就像发生了地震了一样.

彼はまるで前もって結果を知っていたかのように、クールな顔で壇上に上がった。
他就像事先已经知道了结果似的,很冷静地走上台。

注意

包含了”虽然实际上不是这样,但是表现得就像那样”的意思,「〜かのようだ」用于句末结局;「〜かのような」后接名词;「〜かのように」用在句中作状语.

30.〜からいいようなものの

接続　文の普通形＋からいいようなものの

説明　“(因为…)幸好(这次)”

例文

この大事な資料が見つかったからいいようなものの、万が一、紛失したらどうするんだ。
幸好找到了这份重要的资料,万一找不到该怎么办.

火事にならなかったからいいようなものの、これからたばこの火の始末に注意してください。
幸好没有发生火灾,今后要注意熄灭烟头.

今回は事故が起こらなかったからいいようなもののこれからよく注意しないといけないですね。
幸好这次没有发生事故,以后一定要注意.

注意

包含了”虽然这次避免了某种不好的事态的发生,但是并不能掉以轻心”的意思.类似的形式还有「〜からよかったものの」,如「あなたがたまたま確認してくれたからよかったものの、もう少しで原稿の締め切りに間に合わなくなるところだった。(幸好你碰巧帮我确认,否则差一点就要赶不上交稿日期了.)」.

27.〜かねない

接続　動ます形＋かねない

説明　表示可能,”有可能”

例文

A:「新人の山田くん、課長に注意されてもまた同じミスをやり」かねませんね。」
B:「そうですね、あの山田くんだからね。」
A:”那个叫山田的新人,即使被课长提醒过了,还是有可能再犯同样的错误吧”
B:”是啊,因为是山田吗”

そんな荒い運転では、事故を起こしかねない。
开车那么野蛮的话,可能会引发事故.

ここまで来ては、ちょっとした間違いが命取りになりかねない。
事情到了这一步的话,一个小小的错误就可能致命.

注意

相当于「〜かもしれない」,不同的是「〜かねない」一般表示可能带来的负面的,不好的结果.

28.〜かねる

接続　動ます形＋かねる

説明　“很难…”,”难以…”

例文

医療研究者の適性は学力試験だけでは判断しかねるところがあるため、面接試験で最終的な合否の判断を行います。
仅凭学习能力的测试,在有些方面很难判断是否适合做医疗研究者,所以通过面试来作出最终合格与否的判断.

残念ながら、ご提案には賛成しかねます。
很遗憾,我不能赞成您的提案.

ご要望には応じかねますので、ご了承ください。
我们很难满足您的要求,请见谅.

注意

①包含了”即使想做某事也很难做到的意思”.惯用表达有「決めるに決めかねる(难以决定)/見るに見かねて(不忍目睹)」.
②属于书面语,用于礼貌地,委婉地拒绝别人的要求等,如「わかりかねます(不知到)/できかねます(不能)」.
③是用肯定的形式来表示否定的意义.

24.〜かと思えば

接続1　動辞書形・た形/名＋かと思えば

説明1　“原以为…却…”

例文1

九月に入ってやっと涼しくなったかと思えば、また急に暑くなったりして残暑が厳しい。
到了9月,原以为终于要凉快了,却突然变热,秋老虎还很厉害.

息子は学校から帰ってきたかと思えば、またすぐ遊びに出かけていった。
儿子从学校回来了,却又马上出去完了.

注意1

后项是与说话人预想相反的事情

接続2 動辞書形+かと思えば

説明2 表示并列,”既…又…”

例文2

老後を一人で生きることを選ぶ熟年夫婦があるかと思えば、新しい旅立ちに向けて、退職後豪華な旅行にでる夫婦もある。
有的中老年夫妻选择独自一人度过晚年,也有些夫妻打算一同踏上新的旅程—退休后参加豪华旅游.

百歳を越えて生きる人がいるかと思えば、一歳になることもなく、この世を去る赤ん坊もいる。
既有活过百岁的寿星,也有还不到一岁就离开人世的婴儿.

注意2

常与「ある/いる」搭配使用,构成「〜かと思えば〜もある/いる」表示”既有…又有…”.

25.〜か〜ないかのうちに

接続　動辞書形＋か＋動ない形ーない＋かのうちに

説明 表示两件事情几乎同时发生,”刚…就…”,”还没…就…”

例文

太郎は「ご馳走様」と言い終わるか終わらないかのうちに、席を立って自分の部屋に入った。
太郎刚说完”我吃饱了”,就离开位子回到自己的房间了.

息子は夕飯を食べ終わるか終わらないかのうちに、またゲームを始めた。
儿子刚吃完晚饭,就开始打游戏了.

電車に乗るか乗らないかのうちに、ドアがしまってしっまた。
刚乘上电车,门就关了.

注意

①使用同一个动词的肯定和否定形
②前后项的两个动作几乎是同时发生,并且包含了”前项动作甚至还没有完成就发生了后项的动作”的意思.
③用于描述已经发生的事实,后项不能是表示命令,意志,推量,否定的内容,并且只能是过去时.

26.〜か何か

接続　動普通形/名＋か何か

説明 表示泛指,”…什么的”

例文

来週の日曜日は「母の日」なので、母にカーネーションの花束か何かを送ろうと思う。
下周就是母亲节了,我想送妈妈一束康乃馨什么的.

仕事の合間、お茶か何かを飲む。
工作间歇,喝喝茶什么的.

パソコンはウィルスに感染したか何かでフリーズして動かなくなった。
电脑因为中毒什么的死机了.

注意

前项内容并不是特定的.「〜か何か」也可以用「名＋や何か」的形式,表示”…或者…(之类的)”.另外,根据「何」后面具体所用到的动词,可将「何」后面的助词「か」变成「を」或「で」等.

A:「ねえ、見て。この本、ベストセラーだって、買っちゃったよ。」
B:「ああ、それって、最近テレビや何かで話題になっている小説でしょ。」
A:你看,听说这本书是畅销书,我就买了.
B:啊,那本书啊,就是最近在电视呀什么的上面经常被提到的那本小说吧.

23.〜(か)と思うと/(か)思ったら

接続1　動た形＋(か)と思うと/(か)思ったら

説明1　表示事情几乎同时发生,”刚…就…”

例文1

朝、息子が慌てて学校に行ったかと思ったら、すぐ「ケータイ！」と叫びながら戻ってきた。
儿子早上刚匆匆忙忙地出门上学,就边喊着”我的手机”边返回了家里.

急に空が暗くなったかと思うと、激しく雨が降り始めた。
天空刚一暗下来,马上就开始下大雨了.

注意1

① 表示前项的事情发生后,紧接着发生了后项的事情,并且后项多为意料之外的事情,包含说话人惊讶的心情.
② 只能用于描述客观事实或他人的情况,不能用于说话人自身的行为。
③ 用于描述现实的状况,后项不能是表示命令,意志,否定的内容.

接続2 動辞書形＋(か)と思うと/(か)思ったら

説明2 表示结果与说话人预想的相反,”原以为…却…”

例文2

やっと部屋がきれいに片付いたと思ったら、子どもたちが学校から帰ってきてまた散らかし始めた。
原以为终于把房间收拾干净了,可结果孩子们从学校回来之后又开始弄得乱七八糟的.

息子は机に向かって勉強を始めたかと思うと、携帯電話でゲームをやり始めた。
我原以为儿子坐在桌前开始学习了,可结果他却玩起了手机游戏.

やっと仕事が終わったかと思ったら、また新しい仕事を頼まれた。
原本以为工作终于结束了,结果又安排了新的工作.

注意2

后项表示与说话人预想相反的事情.

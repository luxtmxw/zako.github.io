79.〜てまたらない

接続 動て形＋たまらない

　　　イ形て形＋たまらない

　　　ナ形語幹＋でたまらない

説明 “…得受不了””非常…”

例文

海外出張中は、幼い子どもたちにあいたくてたまらなかった。
在国外出差的时候,非常想念年幼的孩子们.

みんなの前で、英語の発音の間違いを指摘されて、恥ずかしくてたまらなかった。
在众人面前被指出了英语发音的错误,真是羞愧难当。

蚊に刺された脚が痒くてたまらない。
脚被蚊子叮了,奇痒难忍.

注意

①与「〜てしょうがない」意思大致相同,强调忍受不了某种感觉.不能接在表示自发的动词或词组之后,如「思える/気になる」.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等.

80.〜てでも

接続 動て形＋でも

説明 表示采取强硬的手段,”就算…也要…”

例文

今日は、徹夜してでもこの仕事を終らせなければならない。
今天就算通宵也要把这个工作做完.

姉は昔から「欲しいものは借金をしてでも買う」タイプの人だ。
我姐姐以前开始就是”想要的东西,就算是借钱也要买的”那种人.

家を売ってでも子どもの手術の費用を用意したい。
就算是卖房子,也要凑齐孩子的手术费用.

注意

表示为了达到某一目的而采取强硬的手段,后项表达了说话人强烈的愿望和意志.

76.〜て以来

接続 動て形＋以来

説明 表示在某个动作之后,某种状态一直持续,”自从…以后”

例文

彼とは五年前に別れて以来、一度も連絡を取っていない。
自从五年前分开以后,就再也没有跟他联系过.

今年は水不足になりそうだ。先月の初めに降って以来、ずっと晴れの日が続いている。
今年可能要雨水不足了.自从上个月初下了雨之后,就一直是晴天。

結婚して以来、仕事と家事に追われる日々が続いている。
自从结婚以后每天都忙于工作和家务.

注意

①表示前项的事情再也没有发生过,或者另一件事情一直持续.后项是表示状态的,不能是表示一次性的动作的内容.
②前项是过去发生的某件事情,不能是刚刚发生不久的事情.

77.〜てからでないと/てからでなければ

接続 動て形＋から出ないと/からでなければ

説明 表示如果不先做前项,就无法做后项.”如果不先…就…”

例文

部長のサインをもらってからでないと、この機密ファイルを送ることは不可能だ。
没有得到部长的签字的话,就不能发送这份机密文件.

その詩が作られた時代背景を勉強してからでないと、詩人の思いを理解するのは難しいだろう。
不学习这首诗创作的时代背景的话,就很难理解诗人的想法.

子どもを寝かせてからでないと、自分の時間を過ごすことができない。
不先把孩子哄睡着的话,就没有自己的时间.

注意

前项是后项的前提条件.

78.〜てしょうがない/てしようがない

接続 動て形＋しょうばない/しようがない

　　　イ形て形＋しょうがない/しようがない

　　　ナ形語幹＋でしょうがない/でしようがない

説明 表示程度很深,”…得不得了””特别…”

例文

昨日、1点差で試合に負けてしまい、悔しくてしょうがなかった。
昨天以一分之差输掉了比赛,特别不甘心.

眠くてしようがない時は、コーヒーを飲んで、また勉強を頑張る。
困得不行的时候,喝杯咖啡,继续努力学习.

試合の最後に逆転されて、残念でしょうがない。
在比赛最后阶段被反超,特别遗憾.

隣の部屋から聞こえてくる音が気になってしょうがない。
很在意隔壁房间传来的声音.

注意

①相当于「〜てしかたがない」。表示某种感情,感觉很强烈,”…得没有办法了”.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等。

73.〜っぽい

接続 動ます形/イ形語幹/名＋っぽい

説明 表示有某种感觉或倾向,”看上去…””感觉…””容易…”

例文

うちの子どもは、よく言えば好奇心旺盛、悪く言えば飽きっぽい性格で、何をやっても三日坊主だ。
我家孩子,往好的方面说是好奇心旺盛,往坏的方面说就是做事情没有常性,无论做什么都是三天打鱼两天晒网.

彼は大学生のくせに、考え方が幼稚で子どもっぽい。
他虽然是大学生,但是想法幼稚,孩子气。

とても高価な腕時計なのに、安っぽい見える。
虽然是很贵的手表,但是看起来却很廉价。

年を取ると、忘れっぽいなるのが普通だ。
一上了年纪,就往往会变得健忘。

注意

①接在名词和イ形容词后面表示具有前面的名词或形容词所表示的性质,接在动词后面表示容易产生该动词所表示的行为。
②「〜っぽい」相当于一个イ形容词,可以活用变形.
③惯用表达有「子どもっぽい(孩子气的)/白っぽい(发白的)/水っぽい(水分过多的)/熱っぽい(感觉有点发烧的)/湿っぽい(潮湿的;阴郁的)/安っぽい(廉价的)/忘れっぽい(健忘的)/怒りっぽい(易怒的)/飽きっぽい(容易厌倦的)」

74.〜つもりだ/つもり

(1)〜つもりだ

接続 動辞書形・ない形ーない＋つもりだ

説明 “打算…”

例文

来年は日本へ旅行に行くつもりだ。
打算明年去日本旅游.

明日の飲み会には行かないつもりだ。
明天的聚餐我不打算去.

病院に行くつもりだったが、急用ができて行けなくなった。
本打算去医院的,后来有急事没去成.

(2)〜つもり

接続 動た形・ている形/ナ形語幹ーな/イ形/名ーの＋つもり

説明 表示主语自己的想法,”自己认为…”

例文

冗談を言ったつもりだったのに彼女を怒らせてしまった。
我自认为是开玩笑,结果却把她惹生气了.

A「吉田くん、ちょっと物事を消極的に考えるところがあるんじゃない」。
B「いやあ、自分ではけっこう楽観的なつもりでいるけどね」。
A:”吉田,你考虑事情是不是有点消极呢”。
B:”没有呀,我觉得自己还是蛮乐观的”。

彼女は自分では美人のつもりだけど、あまりもてない。
她自以为很漂亮,却不怎么受男生欢迎.

注意

表示后项与前项表述的事实不符.第二个例句中的「つもりで」后面加上「いる」,表示处于某种状态.

75.〜つもりで

接続 動た形＋つもりで

説明 “就当是…”

例文

旅行の行ったつもりでお金をためる。
就当是去旅行了,把钱存了起来.

死んだつもりで一生懸命頑張れば希望が見えてくるだろう。
拼死努力的话,总会看到希望的吧。

学生時代に戻ったつもりで、日本語を勉強する。
就当是又回到了学生时代,来学日语。

注意

第二个例句中的「死んだつもりで」是惯用表达,表示”下定决心做某事”.

71.〜つつ/つつも

接続1 動ます形＋つつ

説明1 表示两个动作同时进行,”一边…一边…”

例文1

辞書を引きつつ英字新聞を読んでいる。
一边查字典一边看英文报纸.

資金のことを考えつつ、工事を進めていく。
一面考虑资金的问题,一面去开展工程。

注意1

相当于「〜ながら」,表示进行一个动作的同时,进行另一个动作,比「〜ながら」更书面化.注意此时要用「つつ」,不用「つつも」。

接続2 動ます形＋つつ(も)

説明2 表示逆接,”虽然…但是…”

例文2

タバコは体に悪いと知りつつも、なかなかやめられない。
虽然知道抽烟对身体不好,但是怎么也戒不掉.

週末は、今日こそ勉強しなきゃと思いつつも、いつもだらだらしてしまう。
周末的时候虽然想着今天必须学习了,但总是懒懒散散地浪费时间.

口では「頑張る」と言いつつも、心の中では落ちてもいいと思っていた。
虽然嘴上说”我会努力的”,但心里却想着即使考不上也关系.

注意2

相当于「〜けれども/のに」。表示逆接是「も」可用可不用.

72.〜つつある

接続 動ます形＋つつある

説明 “正在…”

例文

テロ事件の犯人が捕まり、パリ市民の生活は通常に戻りつつある。
恐怖袭击事件的犯人被逮捕了,巴黎市民的生活正在恢复正常.

ガン治療における早期発見の重要性が人々に認識されつつある。
在癌症治疗方面早期发现的重要性正在被人们所认识到。

政府は景気が緩やかに回復しつつあると発表した。
政府宣布经济形势正在平缓复苏。

注意

表示某个动作正在进行,或者某个变化在不断发展.

69.〜ついでに

接続 動辞書形・た形/名ーの＋ついでに

説明 “顺便…”

例文

コンビニに宅配便を頼みに行ったついでに、牛乳を買ってきた。
去便利店寄快递,顺便买了牛奶回来.

買い物のついでに、駅前にオープンしたラーメン屋で食事をすませた。
买东西的时候,顺便到电车站附近新开的拉面店吃了顿饭。

札幌出張のついでに、雪祭りを見物した。
去札幌出差,顺便去参观了冰雪节.

注意

主要是做前项的事情,顺便做后项的事情。「ついでに」也有作为接续词的功能.

70.〜っこない

接続 動ます形＋っこない

説明 表示强烈的否定,”不可能…”

例文

うつ病を経験したことのない人に、そのつらさはわかりっこない。
没有得过抑郁症的人是不可能懂得那种痛苦的.

今から二日でこの本の単語を全部覚えろなんて、できっこないよ。
说什么从现在起两天内把这本书上的单词全部记住,这可是不可能做到的.

1日10時間も勉強するなんて、私にはできっこないよ。
一天学习十个小时,我可做不到.

注意

一般用于口语,表示对某件事发生的可能性进行强烈的否定.

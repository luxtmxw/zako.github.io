109.〜に(は)かかわらず/に(は)かかわりなく

接続 動辞書形・ない形/ナ形/イ形/名＋に(は)かかわらず/に(は)かかわりなく

説明 “不管…””无论…”

例文

子どものお絵描きというのは上手か下手かにかかわらず、描きたいものを自由に書かせれば、それでいいんです。
不管孩子画画好不好,只要能让他们自由地画自己想画的东西,这就够了。

年齢、性別にかかわらず、誰でも個性と能力を発揮できる社会であってほしい。
希望我们的社会可以不论年龄和性别,任何人都发挥自己的个性和能力。

出席するしないにかかわらず、お返事をお願いします。
无论是否出席,都请您回复。

注意

①接名词时,一般接表示天气、性别、年龄等含有种类、程度差异的名词,如「天候/年齢/晴雨/性別/地位/能力」等,表示不管是哪一种,都无所谓.
②接动词、ナ形容词和イ形容词时,一般接在两个表示对立的词的后面,如「する・しない/好き・嫌い/上手・下手/良し・悪し/いい・悪い」,两个词的后面还可以接「か」,如第三个例句也可以说成「出席するかしないか」

110.〜に限って/に限り

(1)〜に限って/に限り

接続 名＋に限って/に限り

説明 表示限定,”只限于…””只有…”

例文

当店では一万円以上ご購入の方の中から先着50名様に限り、5000円分の商品券をプレゼントいたします。
我们将从在本店消费一万日元以上的顾客中,选定前50名顾客赠送5000日元的商品券.

毎週火曜日に限って、映画は半額になります。
只有每周星期二,电影票半价。

今回の応募作品は著作権を侵害しないものに限る。
这次应征的作品不得侵犯他人著作权.

注意

「〜に限って/〜に限り」用在句中中顿;「〜に限る」用于句末

(2)〜に限って

接続 名＋に限って

説明 “唯独…””偏偏”

例文

寝坊して授業に遅れたときに限って、出席を取られたりする。
偏偏在睡了懒觉上课迟到的时候,会被老师点名.

今日は彼女の誕生日、定時に退社しようと思ったのに、こんな日に限って緊急の仕事が入るんだから、困っちゃう。
今天是女朋友的生日,我想准时下班,却偏偏在这样的日子,接到了紧急的工作,让我很为难。

予習していないときに限って、先生に当てられる。
偏偏在没有预习的时候,被老师点到.

注意

包含”其他时候一般都不怎么发生的情况,偏偏在关键时刻发生了”的意思,后项往往是意想不到的事情

107.〜において/における

接続 名＋において/における

説明 表示时间,地点或状况,”在…””在..方面”

例文

このメーカーの鎮痛剤は、即効性において、他社の薬品より優れている。
这个厂家的镇痛剂在速效方面,比其他公司的药品都要好。

30年前においては、民間人の宇宙旅行などは考えられないことだった。
在30年前,普通老百姓想去宇宙旅行是不可想象的事情。

現代社会における経済格差の問題はますます深刻化している。
现代社会中的经济差距问题日益严重。

注意

「〜における」后面接名词.表示地点时相当于「で」,语气比「で」正式.

108.〜に応じて(は)/に応じ/に応じた

接続 名＋に応じて(は)/に応じ/に応じた

説明 “根据…””按照…”

例文

この旅行会社は、客の予算に応じていくつかのツアーコースを用意してくれる。
这家旅游公司根据客户的预算,提供不同的旅游线路.

当ピアノ教室のレッスン時間は、お客様のご都合に応じて変更可能です。
本钢琴培训班的授课时间可以根据学员方便的时间进行变更。

うちの会社は年に二回評価を行って、その評価に応じて昇進が決まる。
我们公司一年两次考核,根据考核结果来决定升职.

注意

表示根据前项,后项会发生变化,包含”如果前项发生变化,后项也随之变化”的意思.「〜に応じて/に応じ」用在句中中顿;「〜に応じた」后面接名词.

105.〜ならともかく

接続 動辞書形/ナ形語幹/イ形/名＋ならともかく

説明 “如果是…则另当别论”

例文

(不動産屋で)
妻「ねえ、このマンションはどう？ちょっと狭いかなあ。」。
夫「うーん、二人ならともかく、これから子どもも生まれてくるし、もうちょっと大きいのを探そう。」
(在房产中介公司)
妻子:”你看,这间公寓怎么样?有点小吧”。
丈夫:”嗯,两个人的话倒没什么,孩子也快要出生了,所以找了间再大一点的吧”。

幼稚園児ならともかく、小学3年生の子どもを肩車するのは無理がある。
幼儿园小朋友的话也就罢了,让小学三年级的孩子骑在肩上的话就太吃力了。

このレストラン、会社の接待ならともかく、自分のお金で行くには高過ぎる。
这家餐厅,如果是公司招待客人去则另当别论了,自己花钱去的话就太贵了。

注意

表示如果是前项的情况,还可以理解或者还情有可原,但是如果是后项的情况就很难接受了.后项提出比前项程度更高的情况.

106.〜にあたって/にあたり

接続 動辞書形/名＋にあたって/にありた

説明 表示在某个重要的时刻,”值此…之际””在…的时候”

例文

高速道路の建設にあたって、地元住民の理解と協力が得られるよう話し合いをした。
在修建高速公路的时候,为了得到当地居民的理解和配合,和他们进行了沟通.

わが社は創業５０周年を迎えるにあたって、新しいロゴマークを制作した。
我们公司在成立50年周年之际,制作了新的商标。

先輩は面接にあたっての注意事項を教えてくれた。
前辈告诉了我一些面试时的注意事项.

注意

「〜にあたって/にあたり」用在句中中顿;「〜にあたっての」后面可接名词.多用于致辞或感谢信等正式的场合.

103.〜ながら(も)

接続 動ます形/ナ形語幹/イ形/副/名＋ながら(も)

説明 表示逆接,”虽然…但是…”

例文

このワンピースはシンプルでありながらも、上品な女性らしさが感じられる。
这条连衣裙虽然款式很简单,但是给人感觉很高雅、有女人味。

宏くんは門限を過ぎると怒られると知りながら、遊びに夢中で帰ろうとしなかった。
小宏知道过了门禁时间才回家会被训斥,但是还是沉迷于游戏,完全没有要回去的意思。

厳正な選考の結果、残念ながら、今回は採用を見送らせていただきます。
经过严格公正的选拔,非常遗憾,这次您没有被录取。

狭いながらも楽しい我がや。
金窝银窝不如自己的稻草窝。

注意

表示并不是完全没有某种可能性.双重否定表示肯定,但并不是正面的肯定,而是消极的肯定,表示有某种可能性.

104.〜なくはない/なくもない

接続 動ない形＋なくはない/なくもない

ナ形語幹＋で＋なくはない/なくもない

イ形ない形＋なくはない/なくもない

名＋が/は＋なくはない/なくもない

説明 表示并不是完全没有某种事情,”不是不…”

例文

息子「ママ、ディズニーランドに行きたい。いい？」。
母親「今度の英語のテストでAをとったら、考えなくもないけどね。」.
儿子:”妈妈,我想去迪士尼乐园,好吗?”
母亲:”下次英语考试你拿到A的话,也不是不考虑”.

佐知子は、ある日突然大学をやめて周りを驚かせたが、彼女の性格を考えると、理解できなくはない。
佐知子有一天突然从大学退学了,让大家都很吃惊,但是从她的性格来看,也并不是不能理解。

結婚するつもりはなくもない、仕事が安定してからのほうがいいだろう。
也不是不打算结婚,只是觉得等工作安定之后再结婚比较好.

注意

接名词时,一般接在表示心情或意志等名词后面,表示并不是完全没有那个意思或心情.

101.〜ないことはない/ないこともない

接続 動ない形＋ないことはない/ないこともない

イ形ない形＋ないことはない/ないこともない

ナ形語幹＋で＋ないことはない/ないこともない

説明 表示有某种可能性”不是不…””有可能…”

例文

A「鈴木さんは、実家が造り酒屋だから、お酒には強いでしょう。」.
B「いやあ、飲めないことはないんですが、あまり強くないですよ。」
A:”铃木,你老家是造酒厂,所以你酒量很好吧”.
B:”没有啊,喝是能喝,但是酒量并不好”.

私も決勝戦で逆転負けたことがありますから、あなたの悔しさがわからないこともありません。
我曾经在决赛中被反超而输了比赛,所以也不是不理解你懊悔的心情。

この服、サイズがちょっと大きいけど、着られないこともない。
这件衣服的尺寸稍微大了些,但是也不是不能穿。

注意

表示并不是完全没有某种可能性.双重否定表示肯定,但并不是正面的肯定,而是消极的肯定,表示有某种可能性.

102.〜ないではない/ないでもない

接続 動ない形＋ないではない/ないでもない

イ形ない形＋ないではない/ないでもない

ナ形語幹＋で＋ないではない/ないでもない

説明 表示并非完全没有,”并非不…””也不是不…”

例文

彼女の言い方には多少問題がある気がしないでもないけど、言っていることは正しいと思う。
我并没有注意到她的说话方式上多少是有些问题的,但是我认为她说的没错.

彼がこの3か月間どれだけ頑張ってきたのかを知っているので、プロジェクトが成功した時のあの喜びがわからないでもない。
我知道他这三个月是怎么努力过来的,所以也不是不理解项目成功时他那种喜悦的心情。

この方法は正しくないではないが、一番簡単な方法ではない。
这个方法不是不对,但不是最简便的方法.

注意

与「〜ないことはない/ないこともない」相同.

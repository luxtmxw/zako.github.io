39.〜気味

接続　動ます形/名＋気味

説明　表示倾向,程度,”稍微…””有点儿…”

例文

最近、仕事が忙しくて疲れ気味だから、週末は家でのんびり過ごすつもりだ。
最近工作很忙,感觉有点累,所以周末打算在家里悠闲地度过.

昨日から風邪気味で、だるくて何もする気になりません。
昨天开始有点感冒,身体乏力,什么都不想干.

最近食べ過ぎのせいか、太り気味だ。
最近大概是吃多了,感觉有点胖了.

注意

一般用于说话人自己的感觉,并且多用于表示不好的事情。

40.〜きり/きりだ

接続1　動辞書形・た形/名＋きり

説明1　表示限定,”只有…”

例文1

赤ちゃんが生まれてから、夫婦二人きりで旅行するなんて不可能だ。
孩子出生以后,就不可能夫妻两个人去旅游了.

一回きりの人生だから、楽しく過ごしたい。
人生只有一次,所以想开心地度过.

今朝の新聞は見出しを読んだきりで、具体的な内容はまだ読んでいない。
今天早上的报纸只看了标题,具体内容还没看.

注意1

表示限定时,相当于「〜だけ」。

接続2 動た形＋きり/きりだ

説明2 表示前项发生以后,就再也没有发生后项.”一…就再没…”

例文2

中川「あれ？森さん、新しい帽子ですか。」
森「いえ、半年前に買ったきり、あまり使わなかったんです。」/
中川:”咦?森,这是你新买的帽子吗”.
森:”不是,自从半年前买了之后,就没有怎么戴过”.

こんな見事な生け花の作品は私は無理ですよ。だって、生け花は10年前に習ったきりですから。
这么漂亮的插花作品我是做不出来的.因为我十年前学过插花之后就再也没有接触过.

高校時代にお世話になった杉本先生には毎年年賀状を出しているけど、卒業式のときにあってきりだ。
每年都会给高中时代关照过我的杉本老师寄贺年卡,不过我们高中毕业典礼之后就再也没有见过面了.

注意2

①表示发生前项之后,其状态一直持续,并且再也没有发生后项.
②「〜きりだ」用于句末结句,暗示之后再也没有发生过前项的事情,「〜きりで」用在句中中顿,表示再也没有发生后项表示的某件事.
③惯用表达有「寝たきり」,表示”卧床不起”.

37.〜から見ると/から見れば/から見て/から見ても

接続　名＋から見ると/から見れば/から見て/から見ても

説明　表示判断的立场,”从…来看”

例文

親世代からみると、今の若者のライフスタイルに理解できないところも多いだろう。
在父母那代人来看,现在的年轻人的生活方式也有很多无法理解的地方吧.

日本舞踊を習い始めて6年になる山田さんは、自分ではまだまだ下手だと言っている。しかし、まだ２か月の私から見れば山田さんの踊りは本当に美しい。
学习日本舞蹈已经6年的山田,说自己跳得还不够好.然而,在刚学习舞蹈两个月的我看来,她跳得真的很美.

過去の実績から見れば、彼は次期社長として妥当だろう。
从过去的实际业绩来看,他应该是很合适做下任社长的吧.

注意

与「〜からいうと」「〜からすると」意思相近,但是「〜からみると」可以直接接在表示人物的名词后面.「〜から見ても」表示”即使从…来看”。

38.〜代わりに

接続　動/ナ形/イ形/名の名詞修飾形＋代わりに

説明1　表示代替,交换

例文1

引っ越しを手伝ってもらったかわりに、食事をごちそうしよう。
你帮我搬家了,(作为回报)我请你吃饭吧.

友達にノートを貸してもらうかわりに、夕飯をごちそうすることにした。
朋友借笔记给我,(作为回报)我决定请他吃完饭.

私が課長の代わりに明日の展覧会に参加する。
我将代替课长出席明天的展览会.

注意1

接名词时,表示代替某人去做某事.接动词时,常常表示”回报,补偿,交换条件”

説明2　表示事物相反的两面,”虽然…但是…”

例文2

この仕事は安定している代わりに、給料があまり高くない。
这份工作虽然稳定,但工资不怎么高.

インターネットの普及でさまざまな情報が手に入る代わりに、人とのコミュニケーションが少なくなった。
由于互联网的普及,人们可以得到各种各样的信息,但与人的交流却变少了.

都市化が進んでいく中で、面倒くさい近所づきあいから解放されたかわりに、子育ての援助者を失っていく。
在城市化不断发展的过程中,人们虽然摆脱了邻里间麻烦的交往,但也失去了养育孩子的援助者.

注意2

表示某个事物既有前项积极地一面,也有后项的消极的一面,而且两者是相互关联的,往往存在一定程度上的因果关系.

34.〜からすると/からすれば

接続　名＋からすると/からすれば

説明　表示判断的依据,”从…来看”

例文

あの人は話し方からすると、関西の出身だろう。
从那个人的口音来看,大概是关西地区的人吧.

メールの内容からすれば、その客さんはまだ商品使い方をわかっていないようだ。
从邮件的内容来看,那位客人好像还没明白产品的使用方法.

今の実力からすると、勝つのはかなり難しい。
从现在的实力来看,取胜是很难的.

注意

与「〜からいうと」用法相同。

35.〜からといって

接続　文の普通形＋からといって

説明　表示转折,”不能因为..就…”,”虽说…但是…”.

例文

いくら若いからといって、徹夜を続けると体を壊しますよ。
虽说年轻,但是持续熬夜的话,会损坏身体的.

ドイツに留学したことがあるからといってドイツ語が流暢に話せるとは限らない。
虽然在德国留过学,但未必就能说出一口流利的德语.

親だからといって、勝手にこどもの人生を決めるのはあまりよくない。
虽然是家长,但是擅自决定孩子的人生是不好的.

注意

相当于「〜だけの理由で」,前项提出一个原因,后项对其进行否定,认为这个原因不成立或不充分

36.〜からには

接続　動/イ形の普通形＋からには

ナ形語幹/名＋である＋からには

説明　“既然…就…”.

例文

やると決めたからには、途中で投げ出したりせず、最後まで終わらせない。
既然决定了要做,就不会中途放弃,想要坚持到最后.

社会人になったからには、仕事を頑張るつまりだ。
既然已经步入社会了,就打算努力工作.

約束したからには、どんなことがあっても守らなければならない。
既然已经约定好了,无论发生什么事情都必须遵守.

注意

与「〜以上は」「〜上は」意思相同

31.〜からいうと/からいえば/からいったら/からいって

接続　名＋からいうと/からいえば/からいったら/からいって

説明　表示评论的立场,”从…来说”

例文

A:「今の若者に一番伝えないことは何でしょうか。」
B:「そうですね。教育者の立場からいうと」、自分らしさを大切に、本当にやりたいことを見つけてほしいですね.
A:”您最想对现在的年轻人说的话是什么?”
B:”嗯,从教育的立场来说,我希望年轻人能保持自己的个性,找到自己真正想要做的事情.”

私の経験からいうと、目の前の仕事にベストを尽くすことが大事だ。
以我的经验来说,尽全力去做眼前的工作是很重要的.

彼女の性格からいうと、この仕事には向いていない。
从性格来说,她不适合这份工作.

注意

表示”站在某个立场来说,来判断”,注意前面不能直接接表示人物的名词.如第三个例句中的「彼女の性格からいうと」不能说成「彼女からいうと」。

32.〜からこそ

接続　文の普通形＋からこそ

説明　强调原因,”正因为…”.

例文

いつでもどこでも読める電子書籍がはやっている時代だからこそ、図書館でじっくり本を読む時間は貴重だ。
当今时代,无论何时何地都能阅读的电子书籍十分流行,正因为如此,坐在图书馆里沉下心阅读的时间是宝贵的.

息子に生活費を自分で稼がせるのは、彼のためだと思っているからこそだ。
让儿子自己赚生活费,正是为他着想.

本当にこの仕事が好きだからこそ、30年も続けてこられたと思います。
我觉得正是因为真的喜欢这份工作,所以才能坚持做了30年.

逆境にあったからこそ、いろいろなことを学び成長できた。
正因为身处逆境,才更能学到东西,得到成长.

注意

①「こそ」表示强调,接在表示原因的「から」后面用来强调这个原因很重要,包含”不是因为别的,就是因为这个原因”的意思,是说话人主观的判断,因此不能用于表示客观地原因.
②一般不用于表示负面的原因.

33.〜からして

接続　名＋からして

説明　表示判断的依据,”单从…来看”.

例文

ここはさすがに老舗に温泉旅館ですね。木造の建物からして、長い歴史が感じられる。
这里不愧是老字号的温泉旅馆.单从木制的建筑物来看,就能感受到其悠久的历史.

村田先生の新作小説は、そのタイトルからしておもしろい。
田村老师新的小说,单从书名来看就很有意思.

このレストラン、雰囲気からして結構高そうだね。
这家餐厅,单从氛围来看就好像很贵.

注意

表示某一事物包含多个因素,仅从其中一个因素去判断整体,包含”别的怎么样先不说,单从这一点看就…”的意思

29.〜かのようだ/かのような/かのように

接続　動普通形＋かのようだ/かのような/かのように

説明　“就像…一样”

例文

夕方の空は、ひが燃えているかのような夕日が一面に広がっていた。
傍晚时分,烈火般的夕阳染红了天空.

あの少女の彫刻はとてもよく作られていて、まるで生きているかのようだ。
那座少女的雕像做得很逼真,就像活的一样.

大きな爆発でビルが揺れ、まるで地震が起こったかのようだった。
大楼因为巨大的爆炸而晃动,简直就像发生了地震了一样.

彼はまるで前もって結果を知っていたかのように、クールな顔で壇上に上がった。
他就像事先已经知道了结果似的,很冷静地走上台。

注意

包含了”虽然实际上不是这样,但是表现得就像那样”的意思,「〜かのようだ」用于句末结局;「〜かのような」后接名词;「〜かのように」用在句中作状语.

30.〜からいいようなものの

接続　文の普通形＋からいいようなものの

説明　“(因为…)幸好(这次)”

例文

この大事な資料が見つかったからいいようなものの、万が一、紛失したらどうするんだ。
幸好找到了这份重要的资料,万一找不到该怎么办.

火事にならなかったからいいようなものの、これからたばこの火の始末に注意してください。
幸好没有发生火灾,今后要注意熄灭烟头.

今回は事故が起こらなかったからいいようなもののこれからよく注意しないといけないですね。
幸好这次没有发生事故,以后一定要注意.

注意

包含了”虽然这次避免了某种不好的事态的发生,但是并不能掉以轻心”的意思.类似的形式还有「〜からよかったものの」,如「あなたがたまたま確認してくれたからよかったものの、もう少しで原稿の締め切りに間に合わなくなるところだった。(幸好你碰巧帮我确认,否则差一点就要赶不上交稿日期了.)」.

126.〜に違いない

接続 動/イ形の普通形＋に違いない

　　　ナ形語幹/名＋に違いない

説明 表示断定,”一定…””肯定…”

例文

いくら探しても携帯電話が見つからない。どこがに落としたに違いない。
不管我怎么找都找不到手机.肯定是掉在什么地方了。

それを聞いて怒り出した鈴木さんは、きっと何か勘違いをしているに違いない。
听完那件事之后铃木勃然大怒,他肯定是误会什么了。

彼は日本に１０年以上住んでいたので、日本語のレベルが高いに違いない。
他在日本生活了1十多年了,日语水平一定很高。

注意

表示说话人根据事实作出非常肯定的判断.与「〜に相違ない」相同.

127.〜につき

接続 名＋につき

説明1 “每…”

例文1

ただ今、こちらの餃子は、ひとパックにつき100円値引きいたします。
现在这边的饺子每盒便宜100日元.

この幼稚園は子ども１０人につき一人の先生が配置されている。
这家幼儿园每10个孩子配备一个老师。

注意1

有”平均分配”的意思.

説明2 表示原因”因为…”

お盆休みにつき、次回の入荷は20日を予定しております。
由于盂兰盆节假期,下次进货安排在了20号.

ただ今水道工事中につき、一週間休ませていただきます。
由于正在维修下水道,暂停营业一个星期。

注意2

是很郑重的表示原因的用法.常用于通知、告示、书信等.

128.〜につけ/につけては/煮付けても

接続1 動辞書形＋につけ/につけては/につけても

説明1 “每当…就…”

例文1

大手家電メーカーが倒産の危機に陥ったらしい。それにつけても今はなんと競争の激しい時代だろう。
据说有大型家电厂商陷入了破产危机.现在真是一个竞争激烈的时代啊.

この写真を見るにつけ、子どもの頃を思い出す。
每当看到这张照片,就想起小时候的事情。

注意1

常与「見る/聞く/考える」等动词搭配使用,后项一般是表示感情或思考等内容.
有「何かにつけ(动不动,每当有什么契机)/何事につけても(无论什么场合)/それにつけても(总之;真是)」这样的惯用表达.

接続2 動辞書形/イ形/名＋につけ＋動辞書形/イ形/名＋につけ

説明2 “无论…还是…都”

学生の成績がいいにつけ、悪いにつけ、先生としては励まし続けるべきだ。
不管学生的成绩是好是坏,作为老师都应该不断激励他们.

雨につけ、雪につけ、あの人は毎日３０分のジョギングを続ける。
无论是下雨还是下雪,那个人度坚持每天慢跑三十分钟。

彼は嬉しいにつけ、悲しいにつけ、よくお酒を飲む。
他不管是高兴还是悲伤,都经常喝酒。

注意2

列举两个具有对立意义的词,表示”无论是哪张情况”,偶尔也能列举并列意义的词.多为惯用表达,使用范围比较小.

123.〜に過ぎない

接続 動/イ形の普通形＋に過ぎない

　　　ナ形語幹ーである＋に過ぎない

　　　名(＋である)＋過ぎない

説明 “只不过…而已”

例文

東山小学校は、開校当時は３０人程度に過ぎなたったが、今では１０００人を超える学生が在籍している。
东山小学在办校之初才30人左右,而现在已经有超过1000名学生在读。

宇宙は多くの謎に包まれている。これまでの研究でわかったことは、ほんの一部に過ぎない。
宇宙中有很多的谜团.在迄今为止的研究中了解到的还只不过是其中的一小部分。

私はボランティア活動の一員に過ぎません。見えないところで社会のために頑張っている人たちは大勢いると思います。
我只不过是志愿者中的一员.我觉得在大家看不到的地方,还有很多人都在为了社会而努力。

注意

表示程度轻,常与「ただ/ほんの」呼应使用

124.〜に相違ない

接続 動/イ形の普通形＋に相違ない

　　　ナ形語幹/名＋に相違ない

説明 表示断定,”一定…””肯定…”

例文

両親共に血液型がO型だから、生まれてくる子どももO型に相違ない。
父母的血型都是O型,所以生出来的孩子也是O型.

この地域の水汚染の問題を根本から解決するのは難しいに相違ない。
从根本上解决这个地区的水污染问题肯定是很难的。

以上、上に申し上げたことは、事実に相違ありません。
以上所述均为事实.

注意

表示说话人非常有把握,常用于书面语,与「〜に違いない」相同.

125.〜に沿って/に沿った

接続 名＋に沿って/に沿った

説明 “沿着…””按照…”

夏の夜は、川に沿ってゆっくり散歩するのが好きだ。
夏天的夜晚,喜欢喜欢沿着河边悠闲散步.

会社の経営方針に沿って、来年度の計画を立てる。
根据公司的经营方针,制定下个年度的计划。

この地域の実態に沿った対策を講じなければならない。
必须符合当地实际情况的对策.

注意

「〜に沿って」用于句中中顿;「〜に沿った」后面接名词.

121.〜にしては

接続 動/イ形の普通形＋にしては

　　　ナ形語幹/名＋にしては

説明 表示评价的标准,”就…而言”

例文

この子は小学生にしては、絵がうまい。
这个孩子还是个小学生,但是画画很好。

初めてオムライスを作ったにしては、上手にできたね。
就第一次做荷包蛋而言,算是做得很不错了。

今住んでいるところは都心にしては、静かで住み心地がいい。
现在住的地方地处市中心算是很安静的了,居住很舒适。

注意

表示后项与前项叙述的事实比相符,包含”事实与必然的预想相反”的意思.用于对他人或物的评价,不能用于说话人自己的事情.

122.〜にしても/にせよ/にしろ

接続1 動/イ形の普通形＋にしても/にせよ/にしろ

　　　ナ形語幹/名(＋である)＋にしても/にせよ/にしろ

説明1 表示让步,”即使…””无论…”

例文1

地元の友達の結婚披露宴に出席するかどうか、いずれにしても早めに返事をしなければいけない。
是否出席老家朋友的婚宴,无论怎样,都必须尽早作出答复.

たった三日の旅行にせよ、準備は必要だ。
即使是三天的旅行,也需要准备一下。

どんなに強そうに見える人にしても、心に弱いところがあるはずだ。
无论是外表看起来多坚强的人,内心也应该有脆弱的地方.

注意1

常与「どんなに/いくら」等词搭配使用,与「〜ても/でも」类似.

接続2 動/イ形の普通形＋にしても/にせよ/にしろ ＋　動/イ形の普通形＋にしても/にせよ/にしろ

　　　ナ形語幹/名(＋である)＋にしても/にせよ/にしろ　＋　ナ形語幹/名(＋である)＋にしても/にせよ/にしろ　

説明2 “无论…还是…”

例文2

今度のスピーチ大会に参加するにしろしないにしろ、早く決めたほうがいい。
无论是否参加这次的演讲比赛,最好快点做决定.

本物にしろ偽物にしろ、実物を見てから買うかどうか決めよう。
不管是真货还是假货,看到实物之后再决定买不买吧。

卒業後就職するにせよ、進学するにせよ、早く決めないといけない。
毕业后,无论是工作,还是继续深造,都必须尽早决定.

注意2

提出具有相反和对立意义的两个事物,表示”无论是哪一方面都…”

117.〜に際して/に際し/に際しての

接続 動辞書形/名＋に際して/に際し/に際しての

説明 “当…之际”,”在…的时候”

例文

当サイトのご利用に際しては、下記の利用条件をご覧いただきご同意の上、ご利用ください。
在使用本网站的,请在阅读并同意下列使用条款之后进行使用。

クレジットカードのお申し込みに際して、こちらの書類にご記入ください。
在申请信用卡的时候,请填写这些文件。

先生から卒業に際してのお別れのことばをいただきました。
老师在毕业时跟我们说了很多临别的话语。

注意

①包含”以某件事情为契机”的伊苏,属于书面语。「〜に際して/に際し」用于句中中顿;「〜に際しての」后面接名词.「与〜に当たって」类似.
②「〜に際して」等同于「〜際は」,如第一个例句中的「ご利用に際して」可以说成「〜ご利用の際は」。「〜際は」前面接「動辞書形/名ーの」,例如「当日の受付の際は、招待状をお出しください。(当天请在签名时出示邀请函.)」。

118.〜に先立って/に先立ち

接続 名＋に先立って/に先立ち

説明 表示时间上的先后,”在…之前””先于…”

例文

研究会の開始に先立って、新しく加わったメンバーが紹介された。
在研讨会开始之前,就介绍了先加入的成员.

間も無く開会いたしますので、お席についてお待ちください。開会に先立ちまして、いくつか注意事項を申し上げます。
大会马上就要开始了,请各位就座等候.在开幕之前,我来讲点注意事项。

映画の一般公開に先立ち、プレミア試写会を行う予定だ。
预定在电影正式上映之前,举行首映礼.

注意

表示在前项之前先做好后项,属于书面语.「〜に先立って/に先立ち」用在句中中顿.

119.〜に従って/に従い

接続 動辞書形/名＋に従って/従い

説明1 “按照…””根据…”

例文1

社長の命令に従って、その商品は生産中止となった。
遵从社长的命令,那款商品停止了生产.

火災発生時は、係員の指示に従って冷静に行動してください。
发生火灾时,请按照乘务员的指令冷静地采取行动。

作業の手順はマニュアルに従って行わなければならない。
工作的流程必须按照操作指南来进行.

注意1

前面经常接表示人、指示、规则等意义的名词,包含”遵从,服从”的意思.「〜に従って/に従い」用在句中中顿.

説明2 “随着…”

例文2

科学技術が進歩するに従って、私たちの生活はますます便利になってきた。
随着科学技术的进步,我们的生活变得越来越便捷了.

海外市場の開拓に従って、我が社の製品はより多く消費者に愛用されている。
随着海外市场的开拓,我们公司的产品得到了更多的消费者的青睐与选购。

インタネットの普及に従って、情報共有化も進んでいる。
随着互联网的普及,信息共享化也不断发展.

注意2

后项随着前项的变化而发生变化.

120.〜にしたら/にすれば/にしても

接続 名＋にしたら/にすれば/にしても

説明 表示立场,”从…立场来看””在…看来”

例文

海外赴任が決まって家族そろってアメリカに行くことになった。当初、妻にしたら「何で私が仕事を辞めて犠牲にならなきゃいけないの」という思いがきっとあっただろう。
去国外赴任的事情已经定了下来了,全家将迁往美国.当初,在妻子看来,肯定有过”为什么非要我辞职作出牺牲呢”这样的想法.

来月、もっと広いマンションに引っ越すことにした。子どもたちにしても、自分の部屋ができて喜ぶだろう。
决定下个月搬到更大的公寓.即使是孩子们,有了自己的房间也会很开心吧。

学生にしたら、宿題は少なければ少ないほどいいだろう。
在学生们看来,作业越少越好吧.

突然父の病気を告げられて、彼女にしたらきっとショックだっただろう。
突然被告知父亲的病情,对他而言一定是个打击吧.

注意

接在表示人物的名词后面,表示站在他人的立场,去猜测他人的想法.注意不能用于表示站在说话人的立场.「〜にしても」表示”即使从…来看”.

114.〜に加えて/に加え

接続 名＋に加えて/に加え

説明 表示添加, “加上…”,”而且…”

例文

夏の記録的な高温に加えて、雨が少なかったため、今年の米の収穫量は大幅に減った。
夏季创纪录的高温,加上降雨稀少,今年大米的收成大幅度减少了。

のどの痛みに加えて、咳や熱も出てきたので会社を休むしかない。
喉咙痛,还咳嗽,发烧,所以只好向公司请假了。

原油の高騰に加えて、農産物も値上がりした。
原油价格暴涨,农作物的价格也上涨了。

注意

「〜に加え」比「〜に加えて」更书面化,两个都用在句中中顿.常常和「〜も」呼应使用,构成「〜に加えて、〜も」。

115.〜に越したことはない

接続 動辞書形/ナ形語幹(である)/イ形＋に越したことはない

名＋である＋に越したことはない

説明　“最好是…””莫过于…””…(的话)就再好不过了”

例文

一泊二日の出張だが、工事現場の視察もあるから、着替えの服は2セット用意するに越したことはない。
虽然是两天一夜的出差,但是要去施工现场视察,所以最好是准备两套换洗衣服.

申請書の提出締め切りは明日の午後四時だが、早めに出せばそれに越したことはない。
提交申请的截止日期是明天下午四点,如果可以提早交的话那就再好不过了。

子どもが無事に生まれてくれればそれに越したことはない。
孩子可以平安出生就再好不过啦.

注意

「それに越したことはない」表示”那样就再好不过了”

116.〜に応えて/に応え

接続　名＋に応えて/に応え

説明 表示响应,报答,”响应…””应…”

例文

お客様のご要望に応えて、来月から営業時間を１時間を延長させていただきます。
应客户的要求,从下个开始将延长一个小时的营业时间.

顧客の声に応えて、入会手続きはウェブでもできるようになりました。
应客户的要求,加入会员的手续费也可以在网上办理了。

当院は、患者さんのご要望に応えて、病状をわかりやすく説明することを心がけています。
本院应患者的要求,平时注意将病情解释的通俗易懂.

両親の期待に応えようとして、必死でピアノの練習をしている。
为了不辜负父母的期待,努力练习钢琴。

注意

常接在「要望/要求/期待」等词的后面,表示响应这些要求或实现这些期待.「〜に応えて/に応え」用于句中中顿.

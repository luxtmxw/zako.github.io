101.〜ないことはない/ないこともない

接続 動ない形＋ないことはない/ないこともない

イ形ない形＋ないことはない/ないこともない

ナ形語幹＋で＋ないことはない/ないこともない

説明 表示有某种可能性”不是不…””有可能…”

例文

A「鈴木さんは、実家が造り酒屋だから、お酒には強いでしょう。」.
B「いやあ、飲めないことはないんですが、あまり強くないですよ。」
A:”铃木,你老家是造酒厂,所以你酒量很好吧”.
B:”没有啊,喝是能喝,但是酒量并不好”.

私も決勝戦で逆転負けたことがありますから、あなたの悔しさがわからないこともありません。
我曾经在决赛中被反超而输了比赛,所以也不是不理解你懊悔的心情。

この服、サイズがちょっと大きいけど、着られないこともない。
这件衣服的尺寸稍微大了些,但是也不是不能穿。

注意

表示并不是完全没有某种可能性.双重否定表示肯定,但并不是正面的肯定,而是消极的肯定,表示有某种可能性.

102.〜ないではない/ないでもない

接続 動ない形＋ないではない/ないでもない

イ形ない形＋ないではない/ないでもない

ナ形語幹＋で＋ないではない/ないでもない

説明 表示并非完全没有,”并非不…””也不是不…”

例文

彼女の言い方には多少問題がある気がしないでもないけど、言っていることは正しいと思う。
我并没有注意到她的说话方式上多少是有些问题的,但是我认为她说的没错.

彼がこの3か月間どれだけ頑張ってきたのかを知っているので、プロジェクトが成功した時のあの喜びがわからないでもない。
我知道他这三个月是怎么努力过来的,所以也不是不理解项目成功时他那种喜悦的心情。

この方法は正しくないではないが、一番簡単な方法ではない。
这个方法不是不对,但不是最简便的方法.

注意

与「〜ないことはない/ないこともない」相同.

99.〜とは限らない

接続 文の普通形＋とは限らない

説明 “不见得…””未必…”

例文

学生時代、成績がよかった人が社会人になっても必ずしも仕事ができるとは限らない。
学生时代成绩好的人,在进入社会后未必工作能力就强.

値段の高いものが、必ずしもいいものだとは限らない。
贵的东西未必就好。

先生だからと言って、何でも知っているとは限らない。
虽说是老师,也未必什么都知道。

注意

①有时和「〜からといって」搭配使用,”虽说…也未必”.
②前面可以接否定形式,构成「〜ないとは限らない」,表示”未必不…”,此时相当于「〜かもしれない」,即”有可能…”.如「しっかりかぎをかけないと、泥棒に入られないとも限らないから注意してください。(不好好锁门的话,也有可能会进小偷,所以要注意.)」

100.〜ないことには

接続 動ない形/イ形ない形＋ないことには

ナ形語幹/名＋でないことには

説明 “如果不…(就不…)”

例文

厳しい現実を受け入れないことには、前に進めない。
如果不接受严峻的现实的话,就无法前进.

一度付き合ってみないことには、結婚相手にふさわしい人かどうか判断できない。
如果不试着交往一下的话,就无法判断那个人是不是适合的结婚对象。

A社が協力してくれないことには、この技術問題は解決できない。
如果没有公司协助的话,这个技术问题是无法解决的.

注意

后项与否定形式呼应使用,前项是后项成立的必要条件,没有前项,就无法实现后项.

96.〜としか言いようがない

接続 文の普通形＋としか言いようがない

ナ形語幹/名＋としか言いようがない

説明 表示强调,”只能说是…”

例文

古代エジプトの人たちがヒラミッドを造ったとは、不思議としか言いようがない。
古埃及人居然建造出了金字塔,只能说是不可思议.

大きな交通事故で軽い怪我で済んだのは幸運としか言いようがない。
在这么重大的交通事故中只受了点轻伤,只能说是幸运。

この病気が治ったことは奇跡だとしか言いようがない。
这种病治好了,只能说是奇迹。

注意

是「〜しか〜ない」和「動ます形＋よう」两个句型的结合,表示”没有别的说法””只能这么说”

97.〜としたら/とすると/とすれば

接続 文の普通形＋としたら/とすると/とすれば

説明 表示假设,”如果…”

例文

仮に1日8時間睡眠をとるとすると、人生の３分の１はベッドで過ごすということになる。
假设每天睡八小时,那么人生有三分之一的时间是在床上度过的.

大勢の人の前で自己紹介をしなければならないとしたら、誰だって少しは緊張するでしょう。
如果必须在众人面前作自我介绍的话,无论是谁都会有点紧张的吧。

1億の宝くじに当たったとしたら、まず何を買いたいですか。
如果中了一亿彩票的话,首先想买什么呢?

締め切りまでにレポートを提出しないとしたら、その科目の単位が取れなくなってしまう。
如果不在截止日期前提交小论文的话,那个科目的学分就拿不到了。

今からバスで行くとすれば、どこで降りればいいですか。
现在乘巴士去的话,要在哪一站下呢?

注意

可以表示假定条件或既定条件.后项可以表示自然而然的结果,判断,推量,疑问等内容.

98.〜とともに

接続 動辞書形/イ形/名＋とともに

ナ形語幹/名＋である＋とともに

説明 表示伴随,”随着…””与…的同时”

例文

時代の移り変わりとともに、日本語はその表記も意味も少しずつ変化してきた。
随着时代的变迁,日语的表记和意思都逐渐发生了变化.

秋が深まるとともに近郊の山々が色づいてきた。
随着秋意渐浓,近郊的山也被点染得色彩斑斓。

自動車の普及とともに交通渋滞の問題も顕在化してきた。
随着汽车的普及,交通堵塞问题也凸显出来了。

注意

表示随着前项的变化,后项也发生变化,或者前项与后项同时发生.前面接名词时,一般是接包含动作意义的名词.

94.〜ところに/ところへ

接続 動辞書形・た形・ている形＋ところに/ところへ

説明 “正当…的时候”

例文

お弁当を持って、ピックニックに出かけようとしたところに、雨が降ってきた。
正当带着便当要去野餐的时候,下起雨来了.

外国に町で、道に迷って困っていたところに、警察が通りかかった。
在国外的城市,正当迷路不知所措时,警察刚好经过。

ちょうど彼女のうわさをしているところへ、本人がやってきた。
正在谈论有关她的传闻时,本人就过来了。

注意

表示正在做某事时发生了另外一件事情,多是出乎意料的事情.惯用表达有「いいところに来ましたね」,表示”你来得正是时候”

95.〜ところを/ところを見ると

(1) 〜ところを

接続 動辞書形・た形・ている形＋ところを

説明 “正当…的时候”

例文

授業中マンガを読んでいるところを先生に見つかった。
上课看漫画的时候,被老师发现了.

犯人は駅前で買い物をしているところを、警察に捕まった。
罪犯正在电车站前面买东西的时候,被警察逮捕了。

注意

后项的动词多是「見られる(被看见)/見つかる(被发现)/捕まる(被抓住)」这些表示看到或发现等意义的动词,对前项叙述的行为有制止或阻挡等意思.

(2)〜ところを見ると

接続 動た形・ている形＋ところを見ると

説明 “从…来看”

例文

彼が自信満々に言っているところを見ると、何か証拠でもつかんでいるようだ。
看他说话自信满满的样子,好像是掌握了什么证据.

母があんなに喜んでいるところを見ると、姉の結婚が決まったようだ。
从妈妈那么高兴得样子来看,好像是姐姐的婚事定下来了。

店の前に長い行列ができているところを見ると、大変人気のラーメ屋のようだ。
从这家店门前排了很长的队来看,好像是一家很受欢迎的拉面店。

注意

前项提出某个事实,说话人根据这一事实作出后项的判断.

91.〜どころか

接続 動/ナ形/イ形の名詞修飾形＋どころか

　　　名＋どころか

説明 表示从根本上进行否定,”哪里…””别说…”

例文

風邪だと思って市販の風邪薬を飲み続けたら、症状はよくなるどころか、悪化してしまった。
以为是感冒,所以一直在吃市面上卖的感冒药,结果病情别说好转了,反而更加恶化了.

足が痛くて、歩くどころか立つのも無理だ。
脚疼,别说走路了,连站都站不了。

この料理は美味しいどころか、辛すぎて全然食べられない。
这个料理哪里好吃啊,太辣了,根本没法吃。

注意

后项内容与前项叙述的事实相反,从根本上否定了前项,并且提出程度更深的后项,”别说…就连…也…”

92.〜ところだった

接続 動辞書形＋ところだった

説明 “差一点就…””险些…”

例文

危ない！今、電柱にぶつかるところだったでしょう。スマホなんか見ないで、ちゃんと歩いて。
危险!你刚刚差点就撞上电线杆了.别看智能手机了,好好走路.

床が濡れていることに気づかず、もう少しで倒れるところだった。
没注意到地板湿了,差点就摔倒了。

気がつくのが遅かったら、危うく火事になるところだったよ。
如果发现晚了,险些就酿成火灾了。

注意

可以用「もう少しで/危うく」等来加强语气,表示”差一点就…”.

93.〜どころではない/どころではなく

接続 動辞書形・ている形/名＋どころではない

説明 表示不是做某事的场合,”不是…的时候””哪能…”

例文

子どもが生まれる前は料理教室に通ったり、映画を見に行ったりしていた。今はそれどころではなく、育児と家事に追われる日々だ。
生孩子之前还上上烹饪班,看看电影.现在哪里有那个闲工夫,每天都忙于带小孩和做家务.

せっかく家族で海に行ったのに、強風で泳ぐどころではなかった。
难得全家人一起去海边玩,却因为风太大根本没办法游泳。

車を買ったばかりでお金がないので、海外旅行どころではない。
刚买了车,没钱,所以根本谈不上去国外旅游。

注意

强调目前没有时间,金钱或精力去做某事.

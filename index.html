66.〜たばかり

接続 動た形＋ばかり

説明 表示刚刚做完某事,”刚…”

例文

このゲーム機は昨日買ってきたばかりなのに、もう弟に壊されてしまった。
这个游戏机是昨天刚买来的,却已经被弟弟弄坏了.

A「ねえ、そのマンガ、読み終わったら貸して」。
B「いいけど、まだ読んでないよ。図書館で借りてきたばかりなんだ」。
A:”哎,那本漫画看完借给我看吧”。
B:”可以啊,不过我还没看完,刚从图书馆借的”。

ここに引っ越したばかりですから道もよくわかりません。
因为刚搬到这里,所以路还不是很熟悉.

出張から帰ってきたばかりで、とても疲れている。
刚刚出差回来,很疲惫.

注意

表示某个动作刚发生完,没有多长时间。

67.〜たびに

接続 動辞書形/名ーの＋たびに

説明 “每当…””每次…”

例文

今住んでいるところは幹線道路に近く、大型車が通るたびに揺れる。
现在住的地方靠近主干道,每次有大型车通过,房屋都会晃动.

この曲を聞くたびに、大学時代のことを思い出す。
每当听到这首曲子,就会想起大学时代.

姉は旅行に行くたびに、旅先から絵葉書を送ってくれる。
姐姐每次出去旅行,都会从当地给我寄明信片.

注意

每当发生前项,就总是会发生后项的事情.「この度」是惯用表达,表示”这次”

68.〜だらけ

接続 名＋だらけ

説明 “满是…””都是…”

例文

小さい頃は、兄たちと泥だらけになって遊ぶのが大好きだった。
小时候特别喜欢喝哥哥们一起玩,玩得浑身是泥.

あいつはお金持ちに見えているが、実は借金だらけだそうだ。
那家伙看起来是有钱人,但据说他负债累累.

こんな間違いだらけの文章はとても読むきにならない。
这种错误百出的文章,实在没心情看下去.

注意

表示”到处都是…””充满了…”,前面的名词一般带有负面意义,表示说话人负面的评价.惯用的搭配有「泥だらけ(沾满泥)/傷だらけ(伤痕累累)/間違いだらけ(错误百出)/借金だらけ(负债累累)/血だらけ(到处是血)/しわだらけ(皱巴巴)」等。

63.〜たところ

接続 動た形＋ところ

説明 表示做了前项的动作之后,发生或发现了后项的结果

例文

化粧品売り場で新製品の問い合わせをしたところ、試用品をもらった。
在化妆品专柜询问了下新产品,他们送了我试用装.

近くの交番に聞いたところ、昨日失くした定期券が届いていた。
去附近的派出所问了下,得知昨天丢失的月票被人捡到送过来了.

彼にお願いしたところ、快く引き受けてくれた。
我拜托了他,他很爽快地答应下来了.

注意

相当于「〜たら」。

64.〜たところが

接続 動た形＋ところが

説明 表示逆接,”然而…””虽然…但是…”

例文

田中さんならわかるだろうと思って聞いてみたところが、彼にもわからないという。
我想田中的话大概会知道的,我就问了问,可他说他也不知道.

今朝は早めに出かけたところが、交通事故に巻き込まれ遅刻してしまった。
今天早上提早出门,可遇到了交通事故,迟到了.

昔住んでいた町を訪ねたところが、様子がすっかり変わっていて迷ってしまった。
去了曾经住过的街道,完全变了样,都不认识了.

注意

表示逆接时,一定要用「〜たところが」的形式.后项的结果是与预想,期待相反.

65.〜たとたん(に)

接続 動た形＋とたん(に)

説明 “一…就…””刚…就…”

例文

改札口を出たとたんに、傘を電車の中に忘れたことに気づいた。
刚出检票口,就发现把伞忘在电车里了.

三日ぶりに会った幼い娘は、私の顔を見たとたん、わあーと泣き出した。
时隔三天,年幼的女儿一看到我,就”哇”的一声哭了起来.

火災報知器のベルが鳴ったとたんに、みんな慌てて外に飛び出した。
火灾报警器一响,人们就慌张往外面跑了出去.

注意

①前项刚发生,接着就发生了后项,后项多是意料之外的事情,并且是说话人亲历的已经发生的事情了,因此句末只能是过去时.
②后项不能是表示命令,意志,推量,否定的内容.

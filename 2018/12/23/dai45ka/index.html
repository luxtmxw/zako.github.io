第４５課

文法

1.动 + ていきます/てきました　(持续/变化)

时间和空间的变化

少子化が進んで、日本の人口はだんだん減っていくでしょう。(少子化的现象日益加剧,日本的人口将会越来越少吧)
二十年間ずっと日本語を勉強してきました。(这20年来我一直在学习日语)

2. 动 + てきました　(出现)

表示某种状态的开始和出现 只限前接非意志动词

ずっと本を読んでいたので、目が疲れてきました。(之前一直在看书,眼睛开始累了.)
お腹が空いてきたんですが、そろそろ食事に行きませんか。　(肚子有点饿了,咱们去吃饭好吗?)

3.小句1 + し, 小句2 + し, 小句3.

表示有多个理由 名词或二类形 + だ ＋ し
只有一个し　暗示还有其他的理由

美味しいし、手軽だし、私は冷凍食品をよく食べます。(好吃又简单,我经常吃冷冻食品)
電気も消えているし、かぎもかかっているし、きっと留守ですよ。(黑着灯,又锁着门,肯定不在家)

注意: “て”形和”~し、~し、~”的区别有两点:
①”て”一般只用于连接形容词和名词,而”~し、~し、~”还可以接动词
②”~し、~し、~”一般用于陈述原因,理由,而”て”形则可用于多个场合

4.动/一类形+ば　动/一类形+ほど+小句

动词ば形+动词基本形+ほど+小句　一类形容词ば形+一类形容词+ほど+小句
越来越怎么怎么样

この本は読めば読むほど、面白いです。(这本书越读越有趣.)
結婚式のスピーチは、短けば短いほどいいと言われていますね。(大家都说结婚典礼上的致辞越短越好.)

二类形容词有时也可以用于这一句型.其接续方式为:”二类形容词+なら＋二类形容词+な＋ほど”或”二类形容词+であれば＋二类形容词+な＋ほど”。前者多用于口语,后者多用于书面语.

子供は元気なら元気なほどいいです。(小孩越健康越好)
子供は元気であれば元気なほどいいです。(小孩越健康越好)

5.”で/へ/から/まで/と” + の

に后面不能加の

空港から市内までのアクセスがよくなりました。(从机场到市区的交通方便了.)
北京から東京への便は1日に１０便以上あります。(一天有10个以上的航班从北京开往东京)
これは母へのプレゼントです。
これは母にのプレゼントです。(X)

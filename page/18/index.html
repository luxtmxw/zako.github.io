11.〜得る

接続　動ます形＋得る

説明　表示可能性,”能……”

例文

地球温暖化が進むと、気候の変化、海面の上昇などの問題が生じ得る。
地球温室效应不断加剧的话,可能会产生气候变化,海平面上升等问题.

今から必死になって頑張れば、志望の大学に入ることも十分にあり得る。
从开始拼命努力的话,考上自己理想的大学还是非常有可能的.

今度のサッカーの決勝戦では、どんなことでも起こりうる。
这次足球决赛,什么样的情况都有可能发生.

注意

①「〜得る」表示可能性,而不是表示能力.而动词的可能态「〜れる/られる」既可以表示可能性,又可以表示能力.如表示”会说中文”时,只能用「中国語が話せる」,而不能用「中国語が話し得る」
②「〜得る」一般用于书面语,其否定形式是「〜得ない」。「あり得る(常有,常见)」比较常用

12.〜得ない

接続　動ます形＋得ない

説明　表示没有某种可能”不太可能…”

例文

他人の夢の中に潜入して、アイディアを盗むなんてありえない話だ。
嵌入别人的梦中,窃取别人的创意什么的,是不可能的事情.

根性がないと、この目標を達しえないでしょう。
没有毅力的话是无法达成这个目标的吧.

株式相場がこれからどう動くか予測し得ない。
股票行情今后将如何变化是无法预测的.

注意

属于书面语,但是「ありえない」在口语中常用.

13.〜おかげで/おかげだ

接続　動/ナ形/イ形/名の名詞修飾形＋おかげで/おかげだ

説明　表示原因,”托…的福”,”多亏…”

例文

中川先生にご指導いただいたおかげで、卒業論文を完成させることができました。
多亏了中川老师的指导,才得以完成了毕业论文.

今日のイベントはスタッフのみなさんのおかげで、とても楽しい時間を過せました。ありがどうございました。
今天的活动多亏了所有工作人员,我们度过了一段非常愉快的时光.非常感谢大家.

現代に生きる私たちは、科学技術のおかげで快適で便利な日常生活を送っている。
由于科技的发展,生活在现代的我们才能过着舒坦便捷的生活.

仕事が予定より早く終わったのは、吉田先輩が手伝ってくれたおかげです。
工作能够提早完成,多亏了吉田前辈的帮忙.

彼が余計なことを言ってくれたおかげで、大変な目に遭った。
就因为他多嘴,我才倒了大霉.

注意

表示由于前项的原因,导致了后项的好结果.偶尔也能表示不好的原因(如第五句),此时是反语用法,包含讽刺的语气.「〜おかげで」用在句中;「〜おかげだ」用在句末结句.

8.〜うちに

接続1　動普通形/イ形　＋　うちに

###　　　　ナ形語幹ーな/名ーの　＋　うちに

説明1　“在……期间”,”趁着……”

例文1

元気なうちに、趣味の山登りを楽しもうと思っている。
我想趁着身体还硬朗,好好地去享受登山这一爱好.

忘れないうちに、手帳にメモしとこう。
趁着还没忘记,先记在记事本上.

若いうちにいろいろなことに挑戦したいと思う。
我想趁着年轻尝试各种挑战.

注意1

「〜うちに」的前面接表示一段时间的表达方式.接动词时,多接「動ている形」和「動ない形ーない」.

接続2　動ている形 和 ない形　＋　うちに

説明2　在某事发生期间,发生了另外的一件事情,可灵活翻译,”……着……”

例文2

何年も飼っているうちに、ワンちゃんは我が家の一員になった。
养了这么多年了,狗狗已经成为了我们家的一员了.

新しい一眼レフを買ったが、１か月も経たないうちに新型が発売されてがっかりした。
我新买了一台单反相机,但是没过一个月就出了新款,很沮丧.

ラグビーはあまり好きではなかったが、何度もテレビで試合を見ているうちに、好きになってきた。
起初并不怎么喜欢橄榄球,但多次在电视机上观看比赛,渐渐地喜欢上了.

生徒たちは何度も話し合っているうちに、自然と仲間意識が出てきた。
学生们经过不断交谈,自然而然地产生了同伴意识.

注意2

相当于「〜ている間に」。「〜うちに」的前项表示持续性的动作,后项表示发生了没有预想到的结果,并且是不知不觉地,自然而然地发生的.

9.〜うちは

接続　動普通形/イ形＋うちは

###　　　　ナ形語幹ーな/名ーの＋うちは

説明　“在(还没)…的时候/期间”

例文

気象庁から大雨警報が発表されているうちは、外出を控えてください。
在气象厅发布大雨警报期间,请尽量不要外出.

周りに米を作る農家が少なってきたが、おじいさんは元気なうちは稲作を頑張りたいと言っている。
周围种植大米的农户越来越少,爷爷却说在身体还硬朗的时候要努力种植水稻.

工事をしているうちは、この道路は通行できません。
在施工期间,这条道路无法通行.

注意

「〜うちは」强调”在某个时间段里,在某个期间”.既可以用于别人的事情,也可以用于说话人自己的事情.接动词时,多接「動ている形」,「動ている形＋うちは」相当于「〜ている間は」。

10.〜(よ)うではないか/(よ)うじゃないか

接続　動意向形＋ではないか/じゃないか

説明　表示建议或劝诱,”让我们…吧”

例文

みなさん、自然環境を守るために、リサイクル活動に協力しようではありませんか。
为了保护环境,让我们一起支持废品再利用运动吧.

今月の売り上げ目標も達成できたし、今日はみんなで大いに飲もうじゃないか。
这个月的销售目标也完成了,今天大家一起痛痛快快地喝一杯吧.

優勝を目指して一緒に頑張ろうじゃないか。
让我们为了胜利一起努力吧.

注意

包含强烈的呼吁,号召的语气,「〜(よ)うではありませんか」是其礼貌的表达方式.

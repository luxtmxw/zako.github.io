94.〜ところに/ところへ

接続 動辞書形・た形・ている形＋ところに/ところへ

説明 “正当…的时候”

例文

お弁当を持って、ピックニックに出かけようとしたところに、雨が降ってきた。
正当带着便当要去野餐的时候,下起雨来了.

外国に町で、道に迷って困っていたところに、警察が通りかかった。
在国外的城市,正当迷路不知所措时,警察刚好经过。

ちょうど彼女のうわさをしているところへ、本人がやってきた。
正在谈论有关她的传闻时,本人就过来了。

注意

表示正在做某事时发生了另外一件事情,多是出乎意料的事情.惯用表达有「いいところに来ましたね」,表示”你来得正是时候”

95.〜ところを/ところを見ると

(1) 〜ところを

接続 動辞書形・た形・ている形＋ところを

説明 “正当…的时候”

例文

授業中マンガを読んでいるところを先生に見つかった。
上课看漫画的时候,被老师发现了.

犯人は駅前で買い物をしているところを、警察に捕まった。
罪犯正在电车站前面买东西的时候,被警察逮捕了。

注意

后项的动词多是「見られる(被看见)/見つかる(被发现)/捕まる(被抓住)」这些表示看到或发现等意义的动词,对前项叙述的行为有制止或阻挡等意思.

(2)〜ところを見ると

接続 動た形・ている形＋ところを見ると

説明 “从…来看”

例文

彼が自信満々に言っているところを見ると、何か証拠でもつかんでいるようだ。
看他说话自信满满的样子,好像是掌握了什么证据.

母があんなに喜んでいるところを見ると、姉の結婚が決まったようだ。
从妈妈那么高兴得样子来看,好像是姐姐的婚事定下来了。

店の前に長い行列ができているところを見ると、大変人気のラーメ屋のようだ。
从这家店门前排了很长的队来看,好像是一家很受欢迎的拉面店。

注意

前项提出某个事实,说话人根据这一事实作出后项的判断.

91.〜どころか

接続 動/ナ形/イ形の名詞修飾形＋どころか

　　　名＋どころか

説明 表示从根本上进行否定,”哪里…””别说…”

例文

風邪だと思って市販の風邪薬を飲み続けたら、症状はよくなるどころか、悪化してしまった。
以为是感冒,所以一直在吃市面上卖的感冒药,结果病情别说好转了,反而更加恶化了.

足が痛くて、歩くどころか立つのも無理だ。
脚疼,别说走路了,连站都站不了。

この料理は美味しいどころか、辛すぎて全然食べられない。
这个料理哪里好吃啊,太辣了,根本没法吃。

注意

后项内容与前项叙述的事实相反,从根本上否定了前项,并且提出程度更深的后项,”别说…就连…也…”

92.〜ところだった

接続 動辞書形＋ところだった

説明 “差一点就…””险些…”

例文

危ない！今、電柱にぶつかるところだったでしょう。スマホなんか見ないで、ちゃんと歩いて。
危险!你刚刚差点就撞上电线杆了.别看智能手机了,好好走路.

床が濡れていることに気づかず、もう少しで倒れるところだった。
没注意到地板湿了,差点就摔倒了。

気がつくのが遅かったら、危うく火事になるところだったよ。
如果发现晚了,险些就酿成火灾了。

注意

可以用「もう少しで/危うく」等来加强语气,表示”差一点就…”.

93.〜どころではない/どころではなく

接続 動辞書形・ている形/名＋どころではない

説明 表示不是做某事的场合,”不是…的时候””哪能…”

例文

子どもが生まれる前は料理教室に通ったり、映画を見に行ったりしていた。今はそれどころではなく、育児と家事に追われる日々だ。
生孩子之前还上上烹饪班,看看电影.现在哪里有那个闲工夫,每天都忙于带小孩和做家务.

せっかく家族で海に行ったのに、強風で泳ぐどころではなかった。
难得全家人一起去海边玩,却因为风太大根本没办法游泳。

車を買ったばかりでお金がないので、海外旅行どころではない。
刚买了车,没钱,所以根本谈不上去国外旅游。

注意

强调目前没有时间,金钱或精力去做某事.

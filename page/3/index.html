79.〜てまたらない

接続 動て形＋たまらない

　　　イ形て形＋たまらない

　　　ナ形語幹＋でたまらない

説明 “…得受不了””非常…”

例文

海外出張中は、幼い子どもたちにあいたくてたまらなかった。
在国外出差的时候,非常想念年幼的孩子们.

みんなの前で、英語の発音の間違いを指摘されて、恥ずかしくてたまらなかった。
在众人面前被指出了英语发音的错误,真是羞愧难当。

蚊に刺された脚が痒くてたまらない。
脚被蚊子叮了,奇痒难忍.

注意

①与「〜てしょうがない」意思大致相同,强调忍受不了某种感觉.不能接在表示自发的动词或词组之后,如「思える/気になる」.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等.

80.〜てでも

接続 動て形＋でも

説明 表示采取强硬的手段,”就算…也要…”

例文

今日は、徹夜してでもこの仕事を終らせなければならない。
今天就算通宵也要把这个工作做完.

姉は昔から「欲しいものは借金をしてでも買う」タイプの人だ。
我姐姐以前开始就是”想要的东西,就算是借钱也要买的”那种人.

家を売ってでも子どもの手術の費用を用意したい。
就算是卖房子,也要凑齐孩子的手术费用.

注意

表示为了达到某一目的而采取强硬的手段,后项表达了说话人强烈的愿望和意志.

76.〜て以来

接続 動て形＋以来

説明 表示在某个动作之后,某种状态一直持续,”自从…以后”

例文

彼とは五年前に別れて以来、一度も連絡を取っていない。
自从五年前分开以后,就再也没有跟他联系过.

今年は水不足になりそうだ。先月の初めに降って以来、ずっと晴れの日が続いている。
今年可能要雨水不足了.自从上个月初下了雨之后,就一直是晴天。

結婚して以来、仕事と家事に追われる日々が続いている。
自从结婚以后每天都忙于工作和家务.

注意

①表示前项的事情再也没有发生过,或者另一件事情一直持续.后项是表示状态的,不能是表示一次性的动作的内容.
②前项是过去发生的某件事情,不能是刚刚发生不久的事情.

77.〜てからでないと/てからでなければ

接続 動て形＋から出ないと/からでなければ

説明 表示如果不先做前项,就无法做后项.”如果不先…就…”

例文

部長のサインをもらってからでないと、この機密ファイルを送ることは不可能だ。
没有得到部长的签字的话,就不能发送这份机密文件.

その詩が作られた時代背景を勉強してからでないと、詩人の思いを理解するのは難しいだろう。
不学习这首诗创作的时代背景的话,就很难理解诗人的想法.

子どもを寝かせてからでないと、自分の時間を過ごすことができない。
不先把孩子哄睡着的话,就没有自己的时间.

注意

前项是后项的前提条件.

78.〜てしょうがない/てしようがない

接続 動て形＋しょうばない/しようがない

　　　イ形て形＋しょうがない/しようがない

　　　ナ形語幹＋でしょうがない/でしようがない

説明 表示程度很深,”…得不得了””特别…”

例文

昨日、1点差で試合に負けてしまい、悔しくてしょうがなかった。
昨天以一分之差输掉了比赛,特别不甘心.

眠くてしようがない時は、コーヒーを飲んで、また勉強を頑張る。
困得不行的时候,喝杯咖啡,继续努力学习.

試合の最後に逆転されて、残念でしょうがない。
在比赛最后阶段被反超,特别遗憾.

隣の部屋から聞こえてくる音が気になってしょうがない。
很在意隔壁房间传来的声音.

注意

①相当于「〜てしかたがない」。表示某种感情,感觉很强烈,”…得没有办法了”.
②只能用于表达说话人的某种感情,感觉.主语是第三人称时,要在句末加上「らしい/そうだ/ようだ」等。

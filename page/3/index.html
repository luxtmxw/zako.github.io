69.〜ついでに

接続 動辞書形・た形/名ーの＋ついでに

説明 “顺便…”

例文

コンビニに宅配便を頼みに行ったついでに、牛乳を買ってきた。
去便利店寄快递,顺便买了牛奶回来.

買い物のついでに、駅前にオープンしたラーメン屋で食事をすませた。
买东西的时候,顺便到电车站附近新开的拉面店吃了顿饭。

札幌出張のついでに、雪祭りを見物した。
去札幌出差,顺便去参观了冰雪节.

注意

主要是做前项的事情,顺便做后项的事情。「ついでに」也有作为接续词的功能.

70.〜っこない

接続 動ます形＋っこない

説明 表示强烈的否定,”不可能…”

例文

うつ病を経験したことのない人に、そのつらさはわかりっこない。
没有得过抑郁症的人是不可能懂得那种痛苦的.

今から二日でこの本の単語を全部覚えろなんて、できっこないよ。
说什么从现在起两天内把这本书上的单词全部记住,这可是不可能做到的.

1日10時間も勉強するなんて、私にはできっこないよ。
一天学习十个小时,我可做不到.

注意

一般用于口语,表示对某件事发生的可能性进行强烈的否定.

66.〜たばかり

接続 動た形＋ばかり

説明 表示刚刚做完某事,”刚…”

例文

このゲーム機は昨日買ってきたばかりなのに、もう弟に壊されてしまった。
这个游戏机是昨天刚买来的,却已经被弟弟弄坏了.

A「ねえ、そのマンガ、読み終わったら貸して」。
B「いいけど、まだ読んでないよ。図書館で借りてきたばかりなんだ」。
A:”哎,那本漫画看完借给我看吧”。
B:”可以啊,不过我还没看完,刚从图书馆借的”。

ここに引っ越したばかりですから道もよくわかりません。
因为刚搬到这里,所以路还不是很熟悉.

出張から帰ってきたばかりで、とても疲れている。
刚刚出差回来,很疲惫.

注意

表示某个动作刚发生完,没有多长时间。

67.〜たびに

接続 動辞書形/名ーの＋たびに

説明 “每当…””每次…”

例文

今住んでいるところは幹線道路に近く、大型車が通るたびに揺れる。
现在住的地方靠近主干道,每次有大型车通过,房屋都会晃动.

この曲を聞くたびに、大学時代のことを思い出す。
每当听到这首曲子,就会想起大学时代.

姉は旅行に行くたびに、旅先から絵葉書を送ってくれる。
姐姐每次出去旅行,都会从当地给我寄明信片.

注意

每当发生前项,就总是会发生后项的事情.「この度」是惯用表达,表示”这次”

68.〜だらけ

接続 名＋だらけ

説明 “满是…””都是…”

例文

小さい頃は、兄たちと泥だらけになって遊ぶのが大好きだった。
小时候特别喜欢喝哥哥们一起玩,玩得浑身是泥.

あいつはお金持ちに見えているが、実は借金だらけだそうだ。
那家伙看起来是有钱人,但据说他负债累累.

こんな間違いだらけの文章はとても読むきにならない。
这种错误百出的文章,实在没心情看下去.

注意

表示”到处都是…””充满了…”,前面的名词一般带有负面意义,表示说话人负面的评价.惯用的搭配有「泥だらけ(沾满泥)/傷だらけ(伤痕累累)/間違いだらけ(错误百出)/借金だらけ(负债累累)/血だらけ(到处是血)/しわだらけ(皱巴巴)」等。

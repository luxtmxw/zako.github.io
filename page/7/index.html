14.〜恐れがある

接続　動辞書形/名ーの＋恐れがある

説明　表示有发生某种不好的事情的可能性,”恐怕会……”,”有……的危险”

例文

明日も大雨が降り続くと、山崩れが起こるおそれがあります。
明天还继续下大雨的话,可能会发生山体滑坡.

交渉の場ではっきり態度を示さないと、相手に誤解されるおそれがある。
在谈判的时,如果态度不明确的话,可能会被对方误解.

このままだど試合の後半に逆転されるおそれがある。
照这样下去,下半场有被逆转的可能.

注意

「〜おそれがある」只能表示不好的事情发生的可能性,相当于「〜危険/心配/不安がある」等.

15.〜甲斐がある/甲斐がない/甲斐もなく

接続　動た形/名ーの＋甲斐がある/甲斐がない/甲斐もなく

説明　“(没)有…的价值,回报”,”没有白…/白…”

例文

2時間も並んで待ってかいがあって、食べたラーメンは最高においしかった。
排队2个小时没有白等,吃到的拉面好吃极了.

必死の説得のかいもなく、田村くんは会社を辞めた。
田村辞职了,我拼命的劝说他都白费了.

希望を持って楽しく生きなければ、生まれた甲斐がない。
如果不抱着希望快乐地活着的话,就白活了.

注意

「かいがなく/かいもなく」是「〜かいがない」在句中的中顿形式.

16.〜がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして

接続　動普通形ーの/名＋がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして

説明　“以…为契机,机会”

例文

ヒット商品を開発したのがきっかけて、わが社は技術開発への投資を増やした。
开发畅销产品为契机,我们公司加大了对技术开发的投入.

半年前、たまたま「効果的なコミュニケーション法」という市民セミナーに参加した。それをきっかけに、毎月市民セミナーに参加するようになった。
半年前,偶然参加了”有效的交流方式”的市民讲座.以此为契机,之后每个月都参加市民讲座.

引っ越しをきっかけに、家具を買い換えた。
以搬家为契机,换了新家具.

注意

「〜がきっかけで/がきっかけになって」有”…成为契机”的含义,「〜をきっかけに/をきっかけとして」有”…成为契机”的含义、两种形式只有细微的差别.

11.〜得る

接続　動ます形＋得る

説明　表示可能性,”能……”

例文

地球温暖化が進むと、気候の変化、海面の上昇などの問題が生じ得る。
地球温室效应不断加剧的话,可能会产生气候变化,海平面上升等问题.

今から必死になって頑張れば、志望の大学に入ることも十分にあり得る。
从开始拼命努力的话,考上自己理想的大学还是非常有可能的.

今度のサッカーの決勝戦では、どんなことでも起こりうる。
这次足球决赛,什么样的情况都有可能发生.

注意

①「〜得る」表示可能性,而不是表示能力.而动词的可能态「〜れる/られる」既可以表示可能性,又可以表示能力.如表示”会说中文”时,只能用「中国語が話せる」,而不能用「中国語が話し得る」
②「〜得る」一般用于书面语,其否定形式是「〜得ない」。「あり得る(常有,常见)」比较常用

12.〜得ない

接続　動ます形＋得ない

説明　表示没有某种可能”不太可能…”

例文

他人の夢の中に潜入して、アイディアを盗むなんてありえない話だ。
嵌入别人的梦中,窃取别人的创意什么的,是不可能的事情.

根性がないと、この目標を達しえないでしょう。
没有毅力的话是无法达成这个目标的吧.

株式相場がこれからどう動くか予測し得ない。
股票行情今后将如何变化是无法预测的.

注意

属于书面语,但是「ありえない」在口语中常用.

13.〜おかげで/おかげだ

接続　動/ナ形/イ形/名の名詞修飾形＋おかげで/おかげだ

説明　表示原因,”托…的福”,”多亏…”

例文

中川先生にご指導いただいたおかげで、卒業論文を完成させることができました。
多亏了中川老师的指导,才得以完成了毕业论文.

今日のイベントはスタッフのみなさんのおかげで、とても楽しい時間を過せました。ありがどうございました。
今天的活动多亏了所有工作人员,我们度过了一段非常愉快的时光.非常感谢大家.

現代に生きる私たちは、科学技術のおかげで快適で便利な日常生活を送っている。
由于科技的发展,生活在现代的我们才能过着舒坦便捷的生活.

仕事が予定より早く終わったのは、吉田先輩が手伝ってくれたおかげです。
工作能够提早完成,多亏了吉田前辈的帮忙.

彼が余計なことを言ってくれたおかげで、大変な目に遭った。
就因为他多嘴,我才倒了大霉.

注意

表示由于前项的原因,导致了后项的好结果.偶尔也能表示不好的原因(如第五句),此时是反语用法,包含讽刺的语气.「〜おかげで」用在句中;「〜おかげだ」用在句末结句.

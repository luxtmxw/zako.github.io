99.〜とは限らない

接続 文の普通形＋とは限らない

説明 “不见得…””未必…”

例文

学生時代、成績がよかった人が社会人になっても必ずしも仕事ができるとは限らない。
学生时代成绩好的人,在进入社会后未必工作能力就强.

値段の高いものが、必ずしもいいものだとは限らない。
贵的东西未必就好。

先生だからと言って、何でも知っているとは限らない。
虽说是老师,也未必什么都知道。

注意

①有时和「〜からといって」搭配使用,”虽说…也未必”.
②前面可以接否定形式,构成「〜ないとは限らない」,表示”未必不…”,此时相当于「〜かもしれない」,即”有可能…”.如「しっかりかぎをかけないと、泥棒に入られないとも限らないから注意してください。(不好好锁门的话,也有可能会进小偷,所以要注意.)」

100.〜ないことには

接続 動ない形/イ形ない形＋ないことには

ナ形語幹/名＋でないことには

説明 “如果不…(就不…)”

例文

厳しい現実を受け入れないことには、前に進めない。
如果不接受严峻的现实的话,就无法前进.

一度付き合ってみないことには、結婚相手にふさわしい人かどうか判断できない。
如果不试着交往一下的话,就无法判断那个人是不是适合的结婚对象。

A社が協力してくれないことには、この技術問題は解決できない。
如果没有公司协助的话,这个技术问题是无法解决的.

注意

后项与否定形式呼应使用,前项是后项成立的必要条件,没有前项,就无法实现后项.

96.〜としか言いようがない

接続 文の普通形＋としか言いようがない

ナ形語幹/名＋としか言いようがない

説明 表示强调,”只能说是…”

例文

古代エジプトの人たちがヒラミッドを造ったとは、不思議としか言いようがない。
古埃及人居然建造出了金字塔,只能说是不可思议.

大きな交通事故で軽い怪我で済んだのは幸運としか言いようがない。
在这么重大的交通事故中只受了点轻伤,只能说是幸运。

この病気が治ったことは奇跡だとしか言いようがない。
这种病治好了,只能说是奇迹。

注意

是「〜しか〜ない」和「動ます形＋よう」两个句型的结合,表示”没有别的说法””只能这么说”

97.〜としたら/とすると/とすれば

接続 文の普通形＋としたら/とすると/とすれば

説明 表示假设,”如果…”

例文

仮に1日8時間睡眠をとるとすると、人生の３分の１はベッドで過ごすということになる。
假设每天睡八小时,那么人生有三分之一的时间是在床上度过的.

大勢の人の前で自己紹介をしなければならないとしたら、誰だって少しは緊張するでしょう。
如果必须在众人面前作自我介绍的话,无论是谁都会有点紧张的吧。

1億の宝くじに当たったとしたら、まず何を買いたいですか。
如果中了一亿彩票的话,首先想买什么呢?

締め切りまでにレポートを提出しないとしたら、その科目の単位が取れなくなってしまう。
如果不在截止日期前提交小论文的话,那个科目的学分就拿不到了。

今からバスで行くとすれば、どこで降りればいいですか。
现在乘巴士去的话,要在哪一站下呢?

注意

可以表示假定条件或既定条件.后项可以表示自然而然的结果,判断,推量,疑问等内容.

98.〜とともに

接続 動辞書形/イ形/名＋とともに

ナ形語幹/名＋である＋とともに

説明 表示伴随,”随着…””与…的同时”

例文

時代の移り変わりとともに、日本語はその表記も意味も少しずつ変化してきた。
随着时代的变迁,日语的表记和意思都逐渐发生了变化.

秋が深まるとともに近郊の山々が色づいてきた。
随着秋意渐浓,近郊的山也被点染得色彩斑斓。

自動車の普及とともに交通渋滞の問題も顕在化してきた。
随着汽车的普及,交通堵塞问题也凸显出来了。

注意

表示随着前项的变化,后项也发生变化,或者前项与后项同时发生.前面接名词时,一般是接包含动作意义的名词.

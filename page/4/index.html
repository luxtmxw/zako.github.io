96.〜としか言いようがない

接続 文の普通形＋としか言いようがない

ナ形語幹/名＋としか言いようがない

説明 表示强调,”只能说是…”

例文

古代エジプトの人たちがヒラミッドを造ったとは、不思議としか言いようがない。
古埃及人居然建造出了金字塔,只能说是不可思议.

大きな交通事故で軽い怪我で済んだのは幸運としか言いようがない。
在这么重大的交通事故中只受了点轻伤,只能说是幸运。

この病気が治ったことは奇跡だとしか言いようがない。
这种病治好了,只能说是奇迹。

注意

是「〜しか〜ない」和「動ます形＋よう」两个句型的结合,表示”没有别的说法””只能这么说”

97.〜としたら/とすると/とすれば

接続 文の普通形＋としたら/とすると/とすれば

説明 表示假设,”如果…”

例文

仮に1日8時間睡眠をとるとすると、人生の３分の１はベッドで過ごすということになる。
假设每天睡八小时,那么人生有三分之一的时间是在床上度过的.

大勢の人の前で自己紹介をしなければならないとしたら、誰だって少しは緊張するでしょう。
如果必须在众人面前作自我介绍的话,无论是谁都会有点紧张的吧。

1億の宝くじに当たったとしたら、まず何を買いたいですか。
如果中了一亿彩票的话,首先想买什么呢?

締め切りまでにレポートを提出しないとしたら、その科目の単位が取れなくなってしまう。
如果不在截止日期前提交小论文的话,那个科目的学分就拿不到了。

今からバスで行くとすれば、どこで降りればいいですか。
现在乘巴士去的话,要在哪一站下呢?

注意

可以表示假定条件或既定条件.后项可以表示自然而然的结果,判断,推量,疑问等内容.

98.〜とともに

接続 動辞書形/イ形/名＋とともに

ナ形語幹/名＋である＋とともに

説明 表示伴随,”随着…””与…的同时”

例文

時代の移り変わりとともに、日本語はその表記も意味も少しずつ変化してきた。
随着时代的变迁,日语的表记和意思都逐渐发生了变化.

秋が深まるとともに近郊の山々が色づいてきた。
随着秋意渐浓,近郊的山也被点染得色彩斑斓。

自動車の普及とともに交通渋滞の問題も顕在化してきた。
随着汽车的普及,交通堵塞问题也凸显出来了。

注意

表示随着前项的变化,后项也发生变化,或者前项与后项同时发生.前面接名词时,一般是接包含动作意义的名词.

94.〜ところに/ところへ

接続 動辞書形・た形・ている形＋ところに/ところへ

説明 “正当…的时候”

例文

お弁当を持って、ピックニックに出かけようとしたところに、雨が降ってきた。
正当带着便当要去野餐的时候,下起雨来了.

外国に町で、道に迷って困っていたところに、警察が通りかかった。
在国外的城市,正当迷路不知所措时,警察刚好经过。

ちょうど彼女のうわさをしているところへ、本人がやってきた。
正在谈论有关她的传闻时,本人就过来了。

注意

表示正在做某事时发生了另外一件事情,多是出乎意料的事情.惯用表达有「いいところに来ましたね」,表示”你来得正是时候”

95.〜ところを/ところを見ると

(1) 〜ところを

接続 動辞書形・た形・ている形＋ところを

説明 “正当…的时候”

例文

授業中マンガを読んでいるところを先生に見つかった。
上课看漫画的时候,被老师发现了.

犯人は駅前で買い物をしているところを、警察に捕まった。
罪犯正在电车站前面买东西的时候,被警察逮捕了。

注意

后项的动词多是「見られる(被看见)/見つかる(被发现)/捕まる(被抓住)」这些表示看到或发现等意义的动词,对前项叙述的行为有制止或阻挡等意思.

(2)〜ところを見ると

接続 動た形・ている形＋ところを見ると

説明 “从…来看”

例文

彼が自信満々に言っているところを見ると、何か証拠でもつかんでいるようだ。
看他说话自信满满的样子,好像是掌握了什么证据.

母があんなに喜んでいるところを見ると、姉の結婚が決まったようだ。
从妈妈那么高兴得样子来看,好像是姐姐的婚事定下来了。

店の前に長い行列ができているところを見ると、大変人気のラーメ屋のようだ。
从这家店门前排了很长的队来看,好像是一家很受欢迎的拉面店。

注意

前项提出某个事实,说话人根据这一事实作出后项的判断.

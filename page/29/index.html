第４５課

文法

1.动 + ていきます/てきました　(持续/变化)

时间和空间的变化

少子化が進んで、日本の人口はだんだん減っていくでしょう。(少子化的现象日益加剧,日本的人口将会越来越少吧)
二十年間ずっと日本語を勉強してきました。(这20年来我一直在学习日语)

2. 动 + てきました　(出现)

表示某种状态的开始和出现 只限前接非意志动词

ずっと本を読んでいたので、目が疲れてきました。(之前一直在看书,眼睛开始累了.)
お腹が空いてきたんですが、そろそろ食事に行きませんか。　(肚子有点饿了,咱们去吃饭好吗?)

3.小句1 + し, 小句2 + し, 小句3.

表示有多个理由 名词或二类形 + だ ＋ し
只有一个し　暗示还有其他的理由

美味しいし、手軽だし、私は冷凍食品をよく食べます。(好吃又简单,我经常吃冷冻食品)
電気も消えているし、かぎもかかっているし、きっと留守ですよ。(黑着灯,又锁着门,肯定不在家)

注意: “て”形和”~し、~し、~”的区别有两点:
①”て”一般只用于连接形容词和名词,而”~し、~し、~”还可以接动词
②”~し、~し、~”一般用于陈述原因,理由,而”て”形则可用于多个场合

4.动/一类形+ば　动/一类形+ほど+小句

动词ば形+动词基本形+ほど+小句　一类形容词ば形+一类形容词+ほど+小句
越来越怎么怎么样

この本は読めば読むほど、面白いです。(这本书越读越有趣.)
結婚式のスピーチは、短けば短いほどいいと言われていますね。(大家都说结婚典礼上的致辞越短越好.)

二类形容词有时也可以用于这一句型.其接续方式为:”二类形容词+なら＋二类形容词+な＋ほど”或”二类形容词+であれば＋二类形容词+な＋ほど”。前者多用于口语,后者多用于书面语.

子供は元気なら元気なほどいいです。(小孩越健康越好)
子供は元気であれば元気なほどいいです。(小孩越健康越好)

5.”で/へ/から/まで/と” + の

に后面不能加の

空港から市内までのアクセスがよくなりました。(从机场到市区的交通方便了.)
北京から東京への便は1日に１０便以上あります。(一天有10个以上的航班从北京开往东京)
これは母へのプレゼントです。
これは母にのプレゼントです。(X)

第４４課

文法

1. 表示推测 (小句简体形/名+の)ようです(みたいです口语形式) 、 (小句简体形)らしいです 和 そうです的区别

①ようです 根据自己感受到的推测(第六感也算) 主观 直接被观察,被推测

玄関のところに誰かいるようです。

風邪を引いたようです。

遠くから何か飛んできます。飛行機のようですね。
ーーそのようですね。
什么东西从远处飞过来了,好像是飞机.—-好像是.

②らしいです 当别人的面不能使用 偏客观事实

電気が消えています。李さんは出かけたらしいです。
电灯关着,小李大概是出去了.

③有时前两者可以共用 如果站在第三者的角度客观地叙述推测的结论时可共用

李さんは出かけたらしいです。
李さんは出かけたようです。

風邪を引いたようです。
風邪をひいたらしいです。(X) 错误

④(动词ます形去掉ます 或 形容词)そうです 除了推测还有 表示样态(33课)

一类形容词去掉い 加 そうです。　二类和名词直接加

このケーキはとても美味しそうです。(样态)
这个蛋糕看上去很好吃.

雨が降りそうです。
好像要下雨

注意点:
前接形容词否定时用”~そうではありません”, 而前接动词否定时则用”~そうにありません”

あまり美味しそうではありません。
雨やめそうにありません。
雨看上去不像要停的样子.

ない　なさそうです
いい　よさそうです

“~そうです”可以不用再句尾,还可以用于名词的动词前面, 用于动词前用”~そうに”,用于名词前用”~そうな”

美味しそうなケーキですね。
子供たちは楽しそうに遊んでいます。

2.小句(简体形)+らしいです 和 小句(简体形)+そうです(32课) 传闻(听说)

基本一样

天気予報によると、明日は雨だそうです。
据天气预报说明天有雨

林さんはお酒が好きらしいですよ。
据说小林喜欢喝酒.

3.动词/一类形/二类型+ 過ぎます　表示 过了(过多)

表示某种动作或事物的性质等超过了正常的量和程度. 其接续:
动词: ます形去掉ます + 過ぎます
一类形: 去掉い + 過ぎます
二类形: 直接 + 過ぎます
　
昼ご飯を食べ過ぎました。
このスープは熱すぎませんか。(你不觉得这个汤太烫了吗?)
この説明は複雑すぎます。 (这个说明太复杂了)

4.一类形 + さ

“一类形 + さ”　构成一种表示某种状态的名词. 方式吧词尾的い改成さ

この暑さはいつまで続くんでしょう。(这种炎热的天气要持续到什么时候呢)
富士山の美しさが心に残っています。(富士山的美留在了我的心底)

部分一类形后续接さ,成为表示大小,长短,轻重的名词

一类形 ~さ
大きい 大きさ　大小
高い 高さ　高度
深い 深さ　深度
長い 長さ　长度
広い 広さ　面积
太い 太さ　粗度
重い 重さ　重量
厚い 厚さ　厚度
速い 速さ　速度

某些二类形也可以后续接さ。但不如一类形用的普遍.例如:
便利さ(方便程度)　複雑さ(复杂程度)
但不能说 きれいさ(X)　ひまさ(X)

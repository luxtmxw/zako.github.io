31.〜からいうと/からいえば/からいったら/からいって

接続　名＋からいうと/からいえば/からいったら/からいって

説明　表示评论的立场,”从…来说”

例文

A:「今の若者に一番伝えないことは何でしょうか。」
B:「そうですね。教育者の立場からいうと」、自分らしさを大切に、本当にやりたいことを見つけてほしいですね.
A:”您最想对现在的年轻人说的话是什么?”
B:”嗯,从教育的立场来说,我希望年轻人能保持自己的个性,找到自己真正想要做的事情.”

私の経験からいうと、目の前の仕事にベストを尽くすことが大事だ。
以我的经验来说,尽全力去做眼前的工作是很重要的.

彼女の性格からいうと、この仕事には向いていない。
从性格来说,她不适合这份工作.

注意

表示”站在某个立场来说,来判断”,注意前面不能直接接表示人物的名词.如第三个例句中的「彼女の性格からいうと」不能说成「彼女からいうと」。

32.〜からこそ

接続　文の普通形＋からこそ

説明　强调原因,”正因为…”.

例文

いつでもどこでも読める電子書籍がはやっている時代だからこそ、図書館でじっくり本を読む時間は貴重だ。
当今时代,无论何时何地都能阅读的电子书籍十分流行,正因为如此,坐在图书馆里沉下心阅读的时间是宝贵的.

息子に生活費を自分で稼がせるのは、彼のためだと思っているからこそだ。
让儿子自己赚生活费,正是为他着想.

本当にこの仕事が好きだからこそ、30年も続けてこられたと思います。
我觉得正是因为真的喜欢这份工作,所以才能坚持做了30年.

逆境にあったからこそ、いろいろなことを学び成長できた。
正因为身处逆境,才更能学到东西,得到成长.

注意

①「こそ」表示强调,接在表示原因的「から」后面用来强调这个原因很重要,包含”不是因为别的,就是因为这个原因”的意思,是说话人主观的判断,因此不能用于表示客观地原因.
②一般不用于表示负面的原因.

33.〜からして

接続　名＋からして

説明　表示判断的依据,”单从…来看”.

例文

ここはさすがに老舗に温泉旅館ですね。木造の建物からして、長い歴史が感じられる。
这里不愧是老字号的温泉旅馆.单从木制的建筑物来看,就能感受到其悠久的历史.

村田先生の新作小説は、そのタイトルからしておもしろい。
田村老师新的小说,单从书名来看就很有意思.

このレストラン、雰囲気からして結構高そうだね。
这家餐厅,单从氛围来看就好像很贵.

注意

表示某一事物包含多个因素,仅从其中一个因素去判断整体,包含”别的怎么样先不说,单从这一点看就…”的意思

29.〜かのようだ/かのような/かのように

接続　動普通形＋かのようだ/かのような/かのように

説明　“就像…一样”

例文

夕方の空は、ひが燃えているかのような夕日が一面に広がっていた。
傍晚时分,烈火般的夕阳染红了天空.

あの少女の彫刻はとてもよく作られていて、まるで生きているかのようだ。
那座少女的雕像做得很逼真,就像活的一样.

大きな爆発でビルが揺れ、まるで地震が起こったかのようだった。
大楼因为巨大的爆炸而晃动,简直就像发生了地震了一样.

彼はまるで前もって結果を知っていたかのように、クールな顔で壇上に上がった。
他就像事先已经知道了结果似的,很冷静地走上台。

注意

包含了”虽然实际上不是这样,但是表现得就像那样”的意思,「〜かのようだ」用于句末结局;「〜かのような」后接名词;「〜かのように」用在句中作状语.

30.〜からいいようなものの

接続　文の普通形＋からいいようなものの

説明　“(因为…)幸好(这次)”

例文

この大事な資料が見つかったからいいようなものの、万が一、紛失したらどうするんだ。
幸好找到了这份重要的资料,万一找不到该怎么办.

火事にならなかったからいいようなものの、これからたばこの火の始末に注意してください。
幸好没有发生火灾,今后要注意熄灭烟头.

今回は事故が起こらなかったからいいようなもののこれからよく注意しないといけないですね。
幸好这次没有发生事故,以后一定要注意.

注意

包含了”虽然这次避免了某种不好的事态的发生,但是并不能掉以轻心”的意思.类似的形式还有「〜からよかったものの」,如「あなたがたまたま確認してくれたからよかったものの、もう少しで原稿の締め切りに間に合わなくなるところだった。(幸好你碰巧帮我确认,否则差一点就要赶不上交稿日期了.)」.

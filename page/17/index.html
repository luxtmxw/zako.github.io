17.〜限り(は)/限りでは/ない限り

(1)限り

接続1　動辞書形・可能形/名ーの＋限り

説明1　表示最大限度,”尽量……”,”竭尽……”

例文1

思いつく限りの方法をすべて試してみたが、だめだった。
所有能想到的方法我都试过了,但是行不通.

できるかぎりの努力はした。あとは結果を待つのみだ。
尽了自己的最大的努力,之后就是等待结果了.

山奥で道に迷い、声の限りに助けを求めた。
在深山里迷路了,竭力的求救.

注意

表示达到了极限,惯用表达有「力の限り(用尽全力)/声の限り/命の限り(拼命地)/見渡す限り(放眼所及)」.

接続2　動辞書形・ている形/イ形＋限り(は)

ナ形語幹＋な/である＋限り(は)

名＋の/である＋限り(は)

説明2　表示条件”只要…就”

例文2

生きているかぎり、うれしいことも悲しいことも必ずある。
只要活着,就一定会有开心的事和悲伤的事.

定年になっても元気な限り、働き続けたいと思っている人が増えている。
即使退休,只要身体硬朗就还想工作的人在不断增加.

医者である限り、患者を助けるために必死に頑張らなければならない。
只要身为医生,就必须为了救助患者而拼命努力.

注意2

表示在前项状态持续的期间,会发生后项的情况或状态,包含”如果前项条件不成立,后项的状态也有可能发生变化”的意思.

(2).〜限り/限りで(は)

接続　動辞書形・た形・ている形＋限りで(は)

説明　表示在某个限定范围之内,”据…所知””据…调查”等.

例文

うちの両親はとても仲がよく、わたしが覚えている限りでは、一度も喧嘩したことがない。
我父母的感情很好,在我的记忆中他们从来没有吵过架.

わたしが知っている限り、彼女はまだ結婚していない。
据我所知,她还没有结婚.

私が聞いた限りでは、双方はまだ合意に達していない。
据我所听到的,他们双方还没有达成一致.

注意

常与「見る/知る/聞く/調べる」等表示认知的动词搭配使用,是说话人留有余地的说法,包含”在…范围内是这样的,但在这个范围之外是什么情况,就不知道了”的意思.

(3).〜ない限り

接続　動ない形＋ない限り

説明　“只要不…,就…”,”除非…否则就…”.

例文

最近、A国株の急落はB国の株式市場にも影響を及ぼしていて、政府が何か有効な対策を立てない限り、今後も下落しつづけるだろう。
最近,A国股价的暴跌也影响到了B国的股市,如果政府不采取有效对策的话,今后还会继续下跌吧.

シングルマザーだから、誰かに頼んで子どもの面倒を見てもらわない限り、長時間勤務は不可能だ。
因为是单亲妈妈,所以如果不拜托别人帮忙照顾孩子的话,就不能长时间工作.

資金の問題を解決しないかぎり、この工事を続けられない。
除非解决资金问题,否则这项工程奖无法继续进行.

注意

相当于「〜なければ」.后项常与否定形式呼应使用,即「〜ない限り　〜ない」,表示”只要不…就不…”.

14.〜恐れがある

接続　動辞書形/名ーの＋恐れがある

説明　表示有发生某种不好的事情的可能性,”恐怕会……”,”有……的危险”

例文

明日も大雨が降り続くと、山崩れが起こるおそれがあります。
明天还继续下大雨的话,可能会发生山体滑坡.

交渉の場ではっきり態度を示さないと、相手に誤解されるおそれがある。
在谈判的时,如果态度不明确的话,可能会被对方误解.

このままだど試合の後半に逆転されるおそれがある。
照这样下去,下半场有被逆转的可能.

注意

「〜おそれがある」只能表示不好的事情发生的可能性,相当于「〜危険/心配/不安がある」等.

15.〜甲斐がある/甲斐がない/甲斐もなく

接続　動た形/名ーの＋甲斐がある/甲斐がない/甲斐もなく

説明　“(没)有…的价值,回报”,”没有白…/白…”

例文

2時間も並んで待ってかいがあって、食べたラーメンは最高においしかった。
排队2个小时没有白等,吃到的拉面好吃极了.

必死の説得のかいもなく、田村くんは会社を辞めた。
田村辞职了,我拼命的劝说他都白费了.

希望を持って楽しく生きなければ、生まれた甲斐がない。
如果不抱着希望快乐地活着的话,就白活了.

注意

「かいがなく/かいもなく」是「〜かいがない」在句中的中顿形式.

16.〜がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして

接続　動普通形ーの/名＋がきっかけで/がきっかけになって/をきっかけに(して)/をきっかけとして

説明　“以…为契机,机会”

例文

ヒット商品を開発したのがきっかけて、わが社は技術開発への投資を増やした。
开发畅销产品为契机,我们公司加大了对技术开发的投入.

半年前、たまたま「効果的なコミュニケーション法」という市民セミナーに参加した。それをきっかけに、毎月市民セミナーに参加するようになった。
半年前,偶然参加了”有效的交流方式”的市民讲座.以此为契机,之后每个月都参加市民讲座.

引っ越しをきっかけに、家具を買い換えた。
以搬家为契机,换了新家具.

注意

「〜がきっかけで/がきっかけになって」有”…成为契机”的含义,「〜をきっかけに/をきっかけとして」有”…成为契机”的含义、两种形式只有细微的差别.

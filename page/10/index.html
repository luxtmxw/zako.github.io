59.〜せいで/せいか/せいだ/せいにする

接続 動/ナ形/イ形/名の名修飾形＋せいで/せいか/せいだ/せいにする

説明 表示原因,”因为…”

例文

今年は二月の気温が低かったせいか、桜の開花が例年より遅い。
今年可能是因为二月的气温比较低,樱花的花期比往年晚一些.

ストレスのせいか、最近夜ぐっすり眠れない日が多い。
也许是因为压力大,最近经常晚上睡不好.

お腹が痛いのはきっと賞味期限が切れた牛乳を飲んだせいだ。
肚子痛肯定是因为喝了过期的牛奶.

自分のミスを人のせいにするのはよくないよ。
将自己的过错归咎于别人是不对的.

注意

①表示由于某个原因导致了不好的,消极的结果
②「〜せいで」用于句中.「〜せいだ」用于句末,常用于句型「〜のは〜せいにだ」。「〜せいか」表示不确定的原因,”可能是因为…”
③句型「〜せいにする」表示”归咎于…””怪罪于(某人)…”

60.〜たいばかりに/ほしいばかりに

接続 動ます形＋たいばかりに

　　 名ーが＋ほしいばかりに

説明 表示原因,”就是因为想”

例文

ブランドバッグを手に入れたいばかりに、夏休みは一生懸命アルバイトをした。
就是因为想买名牌包,暑假才拼命打工的.

好きな歌手のコンサートを見に行きたいばかりに、病気にかこつけて休みを取った。
就是因为想去看喜欢的歌手的演唱会,才会生病为借口请假.

仕事を早く終わらせたいばかりに、昼休みも取らずに頑張った。
就是因为想早点把工作做完,中午不休息努力干活.

注意

表示为了实现某种急切的愿望而不辞辛苦去做某事,或者不择手段去做某事,包含”一心想…”的意思.

57.〜ずにすむ/ないですむ/なくてすむ

接続 動ない形＋ずにすむ

動ない形＋ないですむ

動ない形＋なくてすむ

する→せずにすむ/しないですむ

説明 “不…(也可以解决)””不用…就可以了”

例文

帰国する先輩から自転車を譲ってもらったので、新しいのを買わなくてすんだ。
即将回国的前辈把自行车转让给了我,所以我就不用买新的了.

今朝乗っていた電車が一時運転見合わせになったけど、早めに家を出たので遅刻せずにすんだ。
今天早上乘坐的电车停止运行了一段时间,但是因为提早出门了,所以没有迟到.

今回は風の症状が軽かったので、病院に行かずに済んだ。
这次的感冒症状很轻,所以就不用去医院了.

注意

可以表示”不必做原来预定要做的事情”(如第一个例句和第三个例句),也可以表示”避免了可能会发生的事情”(如第二个例句)

58.〜ずにはいられない/ないではいられない

接続 動ない形＋ずにはいられない

　　 動ない形＋ないではいられない

　　 する→せずにはいられない/しないではいられない

説明 表示迫切的心情,”不由得…””不得不…””不能不”

例文

女性は三十歳を過ぎると、どうも年齢をきにせずにはいらなくなる。
女性过了30岁,就不得不在意起自己的年龄了.

2時間も待たされて、担当スタッフに文句を言わずにはいらなかった。
让我等了两个小时,所以忍不住队对责人抱怨了几句.

彼は自分のことをほうっておいてでも苦境にいる友達を助けないではいられない人だ。
他这个人,即使不管自己的事情也忍不住要去帮助处于困境的朋友.

注意

表示控制不住做某事的心情,如果不做某事,心里会很不舒服
